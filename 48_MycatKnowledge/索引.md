## 核心概念

| 概念 | 说明 |
|------|------|
| 逻辑库 | 虚拟数据库，对多个实际数据库的映射 |
| 逻辑表 | 虚拟表，对多个实际表的映射 |
| 物理库 | MySQL中真实存在的数据库 |
| 物理表 | MySQL中真实存在的表 |
| 拆分键 | 用于分片的字段 |
| 集群 | 多个数据源组成的逻辑节点，提供高可用和负载均衡 |
