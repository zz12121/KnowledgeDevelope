##题库要点（原始材料）
| 特性        | JDK 1.7                           | JDK 1.8                                      |
| --------- | --------------------------------- | -------------------------------------------- |
| **底层结构**​ | **数组 + 链表**，分段锁（`ReentrantLock`）。 | **数组 + 链表 + 红黑树**，锁头节点（`synchronized`+ CAS）。 |
| **锁粒度**​  | 锁住整个 **Segment**（包含多个桶）。          | 只锁住单个**桶**（链表或树的头节点），粒度更小，并发性更好。             |
| **并发度**​  | 由 `Segment`的数量决定，创建后不可改变。         | 并发度理论上最高可达数组的大小，更灵活。                         |
