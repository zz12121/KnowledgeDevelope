##题库要点（原始材料）
- **-XX:NewRatio**：控制新生代与老年代的比例。例如 `-XX:NewRatio=3`表示老年代:新生代 = 3:1，即新生代占堆的 1/4。
- **-XX:SurvivorRatio**：控制 Eden 区与一个 Survivor 区的比例。例如 `-XX:SurvivorRatio=8`表示 Eden:S0:S1 = 8:1:1。
