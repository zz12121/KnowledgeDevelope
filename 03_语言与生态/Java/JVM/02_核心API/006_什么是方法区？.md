##题库要点（原始材料）
方法区也是**各个线程共享的内存区域**，用于存储已被虚拟机加载的**类型信息、常量、静态变量、即时编译器编译后的代码缓存**等数据。它有一个非常重要的部分——**运行时常量池**，用于存放编译期生成的各种字面量和符号引用。
