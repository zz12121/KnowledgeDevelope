##题库要点（原始材料）
- **并发标记与并发清除**：
    - **并发标记**：垃圾收集器在**应用程序线程运行的同时**，遍历对象图来标记哪些对象是存活的。这大大减少了为了标记所需的总停顿时间。
    - **并发清除**：在标记完成后，**并发地**回收垃圾对象占用的内存空间。对于ZGC和Shenandoah，更关键的是“并发整理”，即在回收的同时移动存活对象以压缩堆空间。
