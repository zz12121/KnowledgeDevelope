##题库要点（原始材料）
这两个方法都用于加载类，但有一个关键区别：
- `Class.forName(String name)`：默认会执行类的**初始化**阶段。
- `ClassLoader.loadClass(String name)`：默认只执行**加载**过程，不会进行链接阶段的**解析**和**初始化**，是一个“懒加载”行为。
