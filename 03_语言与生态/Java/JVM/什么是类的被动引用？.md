##题库要点（原始材料）
与主动使用相对，**被动引用不会触发初始化**。例如：
- 通过子类引用父类的静态字段，不会导致子类初始化。
- 通过数组定义来引用类，不会触发此类的初始化。
- 引用一个类的常量（static final）且在编译期就能确定值的常量，不会触发初始化，因为在编译阶段通过常量传播优化，已经将常量值存储到了调用类的常量池中。
