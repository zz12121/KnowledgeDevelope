##题库要点（原始材料）
类加载器不仅仅是简单地将类文件加载进内存。它还遵循**双亲委派模型**：当一个类加载器收到加载请求时，它首先会委托给其父类加载器去尝试加载，只有当父类加载器无法完成时，自己才尝试加载。这种模型**保证了 Java 核心库的安全性和稳定性**，防止用户自定义的类动态替换掉 Java 自有的核心类。
