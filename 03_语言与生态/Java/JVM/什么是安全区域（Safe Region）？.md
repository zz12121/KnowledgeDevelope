##题库要点（原始材料）
指**在一段代码片段中，引用关系不会发生变化**。在这个区域内的任意地方开始 GC 都是安全的。对于处于休眠或阻塞状态的线程，它们无法主动响应 JVM 的中断请求走到安全点，这些线程就位于安全区域内，GC 时可以忽略它们。
