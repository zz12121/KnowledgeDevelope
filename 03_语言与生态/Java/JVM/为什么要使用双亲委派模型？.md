##题库要点（原始材料）
- **安全性**：防止核心 API 被篡改。例如，用户自定义一个 `java.lang.Object`类，由于双亲委派机制，这个请求会最终委派给启动类加载器，而启动类加载器加载的是核心库中的 `Object`类，从而保证了核心库的安全。
- **避免重复加载**：确保了同一个类在 JVM 中只会被加载一次，因为父加载器加载过后，子加载器就不会再加载。
