##题库要点（原始材料）
**安全点**：指在代码执行过程中，**线程的状态是确定的**，并且所有对象引用关系已知的点。JVM 在进行某些操作（如 GC 的根枚举）时，需要暂停所有用户线程（STW），这个暂停必须发生在安全点。循环末尾、方法调用后、异常抛出点等通常会设置安全点。
