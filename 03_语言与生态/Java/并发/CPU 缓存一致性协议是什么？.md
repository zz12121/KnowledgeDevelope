##题库要点（原始材料）
**缓存一致性协议**（如 MESI）是硬件层面解决伪共享背后问题的机制。它通过定义缓存行的状态（修改、独占、共享、无效）和状态转换规则，来维护多核 CPU 缓存数据的一致性。
