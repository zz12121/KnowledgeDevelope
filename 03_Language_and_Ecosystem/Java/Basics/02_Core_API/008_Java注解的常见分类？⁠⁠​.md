# Java 注解的分类有哪些？

## 一句话说明（白话）
有内置注解、元注解、自定义注解。

## 它解决什么问题 / 为什么重要
理解注解用途与作用范围。

## 核心原理（一步步讲清楚）
- 内置：@Override/@Deprecated。
- 元注解：@Retention/@Target 等。
- 自定义：业务/框架注解。

##典型使用场景
- 框架配置。

## 简单例子 /伪代码
```java
@interface MyAnno {}
```

## 常见坑与误区
-误用 Retention。
