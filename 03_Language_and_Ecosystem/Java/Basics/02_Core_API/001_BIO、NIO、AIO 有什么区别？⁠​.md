# BIO、NIO、AIO 是什么区别？

## 一句话说明（白话）
BIO 阻塞、NIO 非阻塞+多路复用、AIO 真异步。

## 它解决什么问题 / 为什么重要
选择合适的IO模型影响吞吐和资源占用。

## 核心原理（一步步讲清楚）
- **BIO**：一个连接一个线程，阻塞等待。
- **NIO**：Selector轮询多个通道。
- **AIO**：回调/事件驱动，不阻塞线程。

##典型使用场景
- BIO：小规模连接。
- NIO：高并发网络服务。
- AIO：高IO密集异步场景。

## 简单例子 /伪代码
- NIO：`Selector.select()`

## 常见坑与误区
-误以为 NIO 就是异步（其实是非阻塞）。
