# 什么是对象的逃逸分析？

## 一句话说明（白话）
逃逸分析用于判断对象是否“跑出方法”，从而优化内存分配。

## 它解决什么问题 / 为什么重要
减少堆分配与 GC，提升性能。

## 核心原理（一步步讲清楚）
- 如果对象不逃逸，可分配在栈上或被标量替换。
- JVM 根据分析结果进行优化。

##典型使用场景
- 高频小对象创建的性能优化。

## 简单例子 /伪代码
- 启动参数 `-XX:+DoEscapeAnalysis`。

## 常见坑与误区
-依赖逃逸分析优化会受JVM参数和版本影响。
