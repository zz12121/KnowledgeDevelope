# final 有哪些用法？

## 一句话说明（白话）
final 用来“定住”：变量不可变、方法不可重写、类不可继承。

## 它解决什么问题 / 为什么重要
提高代码安全性和可读性，防止不必要的修改或扩展。

## 核心原理（一步步讲清楚）
- **final变量**：只能赋值一次（基本类型值不变，对象引用不变）。
- **final 方法**：禁止子类重写，保证行为稳定。
- **final 类**：禁止继承，防止扩展（如 String）。

##典型使用场景
-常量定义（配合 static）。
-不希望被继承的工具类。
-关键方法防止重写。

## 简单例子 /伪代码
```java
public final class Utils {}
final int a =10;
```

## 常见坑与误区
- final 引用不代表对象不可变。
-误以为 final 能解决并发问题。
