# Java语言有哪些特点？

## 一句话说明（白话）
Java 是面向对象、跨平台、安全且生态完善的通用语言。

## 它解决什么问题 / 为什么重要
帮助理解 Java适用的应用场景和设计理念。

## 核心原理（一步步讲清楚）
- **跨平台**：编译为字节码，JVM负责运行。
- **面向对象**：封装、继承、多态。
- **安全与健壮**：类型检查、异常机制、垃圾回收。
- **多线程支持**：语言层面支持并发。

##典型使用场景
-企业级后端服务。
-大规模分布式系统。
-移动端（Android）与工具链。

## 简单例子 /伪代码
```java
class User { private String name; }
```

## 常见坑与误区
-误以为 Java “慢”，忽视 JIT 优化。
-忽略 JVM 配置导致性能问题。
