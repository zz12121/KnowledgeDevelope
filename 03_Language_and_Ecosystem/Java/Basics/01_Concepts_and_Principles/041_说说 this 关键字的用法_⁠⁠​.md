# super关键字的用法是什么？

## 一句话说明（白话）
super 用于访问父类成员或构造器。

## 它解决什么问题 / 为什么重要
解决子类与父类成员同名或构造器调用需求。

## 核心原理（一步步讲清楚）
- `super.xxx`访问父类字段/方法。
- `super()` 调用父类构造器，必须在构造器首行。

##典型使用场景
-重写方法中复用父类逻辑。

## 简单例子 /伪代码
```java
class B extends A { B(){ super(); } }
```

## 常见坑与误区
- 在构造器中 `this()` 与 `super()`只能写一个。
