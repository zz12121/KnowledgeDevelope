# 一个空 Object 对象占多大空间？

## 一句话说明（白话）
通常16 字节（64位、开启压缩指针时常见）。

## 它解决什么问题 / 为什么重要
理解对象内存开销，有助于性能与内存优化。

## 核心原理（一步步讲清楚）
- 对象头（MarkWord + Klass Pointer）。
- 对齐填充到8 字节倍数。
-具体大小依赖 JVM 配置（压缩指针、对象对齐）。

##典型使用场景
-内存分析与优化。

## 简单例子 /伪代码
- 使用 `jol` 工具查看对象布局。

## 常见坑与误区
-忽略对象头开销，低估内存。

##关联知识
- 对象头
- 压缩指针（CompressedOops）
