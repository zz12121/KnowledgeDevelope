# Java 自动装箱与拆箱是什么？

## 一句话说明（白话）
基本类型与包装类型之间的自动转换。

## 它解决什么问题 / 为什么重要
提升代码简洁性，但可能带来性能和 NPE 风险。

## 核心原理（一步步讲清楚）
- 编译期插入 `valueOf()` 和 `xxxValue()`。
- 拆箱遇到 null 会抛 NPE。

##典型使用场景
- 集合操作（只能放对象）。

## 简单例子 /伪代码
```java
Integer i =1; // 装箱
int j = i; // 拆箱
```

## 常见坑与误区
-频繁装箱导致性能损耗。
- null 拆箱 NPE。
