# 什么是构造函数？构造函数有什么特点？

## 一句话说明（白话）
构造函数是创建对象时执行的初始化方法。

## 它解决什么问题 / 为什么重要
确保对象在创建后处于正确状态。

## 核心原理（一步步讲清楚）
- 名称与类名相同、无返回值。
- new 时调用，可重载。

##典型使用场景
-初始化成员变量、校验参数。

## 简单例子 /伪代码
```java
class User { User(String n){ this.name=n; } }
```

## 常见坑与误区
- 忘了写构造函数导致默认构造器缺失。
