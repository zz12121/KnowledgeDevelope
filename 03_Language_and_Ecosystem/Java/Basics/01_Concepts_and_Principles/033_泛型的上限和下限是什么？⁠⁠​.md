# 泛型的上限和下限是什么？

## 一句话说明（白话）
`? extends` 是上限，`? super` 是下限。

## 它解决什么问题 / 为什么重要
控制泛型类型边界，提升类型安全。

## 核心原理（一步步讲清楚）
- `? extends T`只能读（生产者）。
- `? super T`只能写（消费者）。

##典型使用场景
- 集合 API设计。

## 简单例子 /伪代码
```java
List<? extends Number> a;
List<? super Integer> b;
```

## 常见坑与误区
- 忽略 PECS 原则。
