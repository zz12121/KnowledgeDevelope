# 什么是零拷贝技术？

## 一句话说明（白话）
零拷贝是减少数据在内核/用户态之间复制的技术。

## 它解决什么问题 / 为什么重要
提升 IO 性能，降低 CPU 消耗。

## 核心原理（一步步讲清楚）
- 使用 DMA + 内核缓冲区，减少用户态拷贝。
- Java 可用 `FileChannel.transferTo/From`。

##典型使用场景
- 文件传输、日志系统、高性能网络。

## 简单例子 /伪代码
```java
fileChannel.transferTo(0, size, socketChannel);
```

## 常见坑与误区
- 不同平台/文件系统支持差异。

##关联知识
- NIO
- FileChannel
