#说说你对JDK、JRE、JVM的理解

## 一句话说明（白话）
JDK=开发工具+JRE，JRE=JVM+核心类库。

## 它解决什么问题 / 为什么重要
理解运行环境和开发环境的区别。

## 核心原理（一步步讲清楚）
- JVM：运行字节码。
- JRE：JVM + 标准库。
- JDK：JRE + 编译/调试等工具。

##典型使用场景
-只运行程序需要 JRE。
- 开发需要 JDK。

## 简单例子 /伪代码
- javac/javadoc 属于 JDK。

## 常见坑与误区
- 把JRE当JDK用。
