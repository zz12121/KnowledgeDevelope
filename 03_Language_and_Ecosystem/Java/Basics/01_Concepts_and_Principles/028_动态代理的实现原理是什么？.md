# 动态代理的实现原理是什么？

## 一句话说明（白话）
动态代理在运行时生成代理类，自动包装目标对象。

## 它解决什么问题 / 为什么重要
实现 AOP、日志、事务等通用增强。

## 核心原理（一步步讲清楚）
- JDK代理：基于接口 +反射。
- CGLIB：生成子类字节码。

##典型使用场景
- Spring AOP、RPC。

## 简单例子 /伪代码
```java
Proxy.newProxyInstance(loader, interfaces, handler);
```

## 常见坑与误区
- JDK代理无法代理类。
- CGLIB不能代理 final。
