# JDK1.8 有哪些新特性？

## 一句话说明（白话）
Java8 最大变化是引入函数式编程能力和更现代的日期时间 API。

## 它解决什么问题 / 为什么重要
提升代码可读性与表达力，改善日期时间处理的易错问题。

## 核心原理（一步步讲清楚）
- **Lambda/方法引用**：把行为当作参数传递。
- **Stream**：链式操作集合数据，支持并行。
- **接口默认方法**：接口可演进而不破坏实现类。
- **java.time**：不可变、线程安全的日期时间模型。

##典型使用场景
-集合处理、过滤、聚合。
-并行计算、批量数据操作。
-日期时间计算与格式化。

## 简单例子 /伪代码
```java
List<String> names = list.stream()
 .filter(s -> s.startsWith("A"))
 .map(String::toUpperCase)
 .collect(Collectors.toList());
```

## 常见坑与误区
- Stream 滥用并行导致性能变差。
- 忽略 `Optional`，反而增加空指针。

##关联知识
