# 熟悉 JDK 哪些包？

## 一句话说明（白话）
常用包覆盖集合、并发、IO、网络等核心能力。

## 它解决什么问题 / 为什么重要
展示Java基础掌握面。

## 核心原理（一步步讲清楚）
- `java.lang`：基础类（String、Object）。
- `java.util`：集合、工具。
- `java.io` / `java.nio`：IO。
- `java.util.concurrent`：并发。
- `java.net`：网络。

##典型使用场景
- 日常开发必备。

## 简单例子 /伪代码
- 使用 `List`、`Map`、`Executor`。

## 常见坑与误区
-只熟 API，不懂底层原理。

- 并发包
