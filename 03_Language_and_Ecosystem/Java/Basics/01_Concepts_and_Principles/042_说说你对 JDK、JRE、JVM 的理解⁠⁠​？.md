# this关键字的用法是什么？

## 一句话说明（白话）
this 指向当前对象。

## 它解决什么问题 / 为什么重要
区分成员变量与参数、调用构造器。

## 核心原理（一步步讲清楚）
- `this.xxx`访问当前对象成员。
- `this()` 调用本类其他构造器（首行）。

##典型使用场景
-构造器链调用。
-参数与字段同名时。

## 简单例子 /伪代码
```java
this.name = name;
this();
```

## 常见坑与误区
- 构造器中 this()/super()不能同时出现。
