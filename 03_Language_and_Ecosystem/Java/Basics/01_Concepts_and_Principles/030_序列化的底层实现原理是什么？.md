# 序列化的底层实现原理是什么？

## 一句话说明（白话）
序列化是把对象转换成字节流，便于存储/传输。

## 它解决什么问题 / 为什么重要
让对象能跨进程或持久化。

## 核心原理（一步步讲清楚）
- 实现 `Serializable` 接口。
- JVM 将对象图写成字节流。
-反序列化恢复对象。

##典型使用场景
- 缓存、RPC、消息传输。

## 简单例子 /伪代码
```java
ObjectOutputStream.writeObject(obj);
```

## 常见坑与误区
- serialVersionUID 不一致导致失败。
