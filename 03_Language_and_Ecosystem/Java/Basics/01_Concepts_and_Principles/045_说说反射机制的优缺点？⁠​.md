#说说你对 equals 与 == 的理解

## 一句话说明（白话）
== 比较引用或基本值，equals 比较内容（可重写）。

## 它解决什么问题 / 为什么重要
避免对象比较出错，影响集合逻辑。

## 核心原理（一步步讲清楚）
- 基本类型：== 比较值。
- 引用类型：== 比较地址。
- equals 可重写定义“内容相等”。

##典型使用场景
- 字符串、实体对象比较。

## 简单例子 /伪代码
```java
"a".equals(b);
```

## 常见坑与误区
- 忘记重写 equals/hashCode。
