#说说你对Java反射的理解

## 一句话说明（白话）
反射让程序在运行时获取类信息并操作对象。

## 它解决什么问题 / 为什么重要
支撑框架、配置驱动与插件化。

## 核心原理（一步步讲清楚）
-通过 `Class` 获取方法/字段/构造器。
-运行时动态调用。

##典型使用场景
- Spring/ORM。
- 序列化、注解处理。

## 简单例子 /伪代码
```java
Class<?> c = Class.forName("com.demo.User");
```

## 常见坑与误区
-反射性能开销大。
-破坏封装。

##关联知识
- 注解
- ClassLoader
