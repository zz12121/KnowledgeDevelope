# 什么是字节码？字节码的好处是什么？

## 一句话说明（白话）
字节码是 Java 编译后的中间表示，交给 JVM运行。

## 它解决什么问题 / 为什么重要
实现跨平台和运行时优化（JIT）。

## 核心原理（一步步讲清楚）
- Java 源码编译为 `.class` 字节码。
- JVM解释或 JIT 编译执行。

##典型使用场景
-跨平台部署。
-动态加载/热更新。

## 简单例子 /伪代码
- `javac`生成 `.class` 文件。

## 常见坑与误区
-认为字节码就是机器码（不是）。

- JVM
- JIT
