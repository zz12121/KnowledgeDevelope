#反射机制的应用场景有哪些？

## 一句话说明（白话）
反射让程序在运行时获取类信息并操作对象。

## 它解决什么问题 / 为什么重要
实现通用框架、插件化、配置驱动。

## 核心原理（一步步讲清楚）
-通过 `Class`、`Method`、`Field` 动态访问。
-运行时根据配置执行不同逻辑。

##典型使用场景
- Spring/ORM。
-序列化/反序列化。
-插件系统。

## 简单例子 /伪代码
```java
Class.forName("com.demo.User");
```

## 常见坑与误区
-反射性能低，滥用会慢。
-破坏封装。

##关联知识
- ClassLoader
- 注解
