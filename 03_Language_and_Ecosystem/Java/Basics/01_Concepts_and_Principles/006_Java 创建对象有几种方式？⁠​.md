# Java 创建对象有几种方式？

## 一句话说明（白话）
常见方式是 `new`，高级用法还有反射、clone、反序列化。

## 它解决什么问题 / 为什么重要
理解对象创建手段，便于读懂框架和序列化逻辑。

## 核心原理（一步步讲清楚）
1. **new**：最直接，调用构造器。
2. **反射**：通过 `Class/Constructor` 动态创建。
3. **clone**：对象复制，需实现 `Cloneable`。
4. **反序列化**：从字节流还原对象。

##典型使用场景
-框架实例化（Spring/ORM）。
-对象复制与快照。
-网络传输对象恢复。

## 简单例子 /伪代码
```java
MyObj o1 = new MyObj();
MyObj o2 = MyObj.class.getDeclaredConstructor().newInstance();
```

## 常见坑与误区
-反射创建忽略构造器权限。
-clone 浅拷贝导致共享引用。
