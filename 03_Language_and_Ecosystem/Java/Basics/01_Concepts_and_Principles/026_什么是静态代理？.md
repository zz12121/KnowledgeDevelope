# 什么是静态代理？

## 一句话说明（白话）
静态代理是在编译期写死代理类，手动增强目标类。

## 它解决什么问题 / 为什么重要
实现日志、权限等横切逻辑，但代码量较大。

## 核心原理（一步步讲清楚）
-代理类实现同一接口，内部持有目标对象。
- 调用前后加入增强逻辑。

##典型使用场景
- 简单 AOP需求。

## 简单例子 /伪代码
```java
class Proxy implements Service { private Service target; }
```

## 常见坑与误区
-代理类爆炸（每个类一份）。

##关联知识
- 动态代理
