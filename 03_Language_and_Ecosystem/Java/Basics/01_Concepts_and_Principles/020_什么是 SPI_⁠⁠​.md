# 什么是 SPI？

## 一句话说明（白话）
SPI 是 Java 的服务发现机制，用于实现插件式扩展。

## 它解决什么问题 / 为什么重要
让接口与实现解耦，新增实现无需改代码。

## 核心原理（一步步讲清楚）
- 在 `META-INF/services/接口全名` 声明实现类。
- `ServiceLoader` 扫描并加载实现。

##典型使用场景
- JDBC 驱动加载。
- 日志/序列化框架扩展。

## 简单例子 /伪代码
```java
ServiceLoader<MyService> loader = ServiceLoader.load(MyService.class);
```

## 常见坑与误区
- 配置文件路径/命名错误导致加载失败。
