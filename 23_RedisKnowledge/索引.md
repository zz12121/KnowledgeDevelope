### [[11_实战场景/01、常见实战场景实现|11. 实战场景]]

| 场景 | 数据结构 | 核心思路 |
|------|----------|----------|
| 限流 | String / ZSet | INCR 计数 / 滑动窗口 / 令牌桶 |
| 排行榜 | ZSet | ZADD + ZREVRANGE + ZREVRANK |
| 签到统计 | Bitmap | SETBIT / BITCOUNT |
| 点赞 | Hash + Set | hincrBy 计数 + SISMEMBER 防重 |
| 共同好友 | Set | SINTER 集合交集 |
| 附近的人 | GEO | GEOADD + GEORADIUS |
| 购物车 | Hash | field=productId / value=数量 |
| 秒杀 | String + Set | Lua 原子扣减 + SADD 防重 |
| 分布式 Session | Hash | sessionId → Hash 字段 |
| 唯一 ID | String | INCR / INCRBY 号段模式 |
| 推荐系统 | ZSet | 协同过滤 + 相似度排序 |
| 搜索补全 | ZSet | ZRANGEBYLEX 前缀匹配 |
| 社交关系链 | ZSet | ZINTERSTORE 求共同关注 |
