# 观察者模式（Observer）

> 定义一种**一对多的依赖关系**，当主题（被观察者）状态改变时，所有观察者自动收到通知并更新。

## 故事引入

### 《同学们，交作业啦！》

小明写完作业后，想通过微信提交给老师。但老师不在线，小明每隔一会儿就看看老师在线没——这可太累了！

后来老师想了个办法："大家做完作业就忙自己的事吧，我要批作业时会在群里通知大家，然后你们再提交。"

这样小明就不用一直盯着微信了，等老师通知就行。

**观察者模式**：定义了对象之间的一对多依赖，当一个对象改变状态时，所有依赖者都会收到通知。

## 二、Demo

```java
// 原生观察者实现（不依赖 Spring）
public interface EventListener<T> {
    void onEvent(T event);
}

public class EventPublisher<T> {
    private final List<EventListener<T>> listeners = new CopyOnWriteArrayList<>();

    public void subscribe(EventListener<T> listener) { listeners.add(listener); }
    public void unsubscribe(EventListener<T> listener) { listeners.remove(listener); }

    public void publish(T event) {
        listeners.forEach(listener -> {
            try {
                listener.onEvent(event);
            } catch (Exception e) {
                System.err.println("监听器处理异常: " + e.getMessage());
            }
        });
    }
}

// 领域事件
public class StockUpdatedEvent {
    private final Long productId;
    private final int oldStock;
    private final int newStock;
    private final LocalDateTime occurredAt;

    public StockUpdatedEvent(Long productId, int oldStock, int newStock) {
        this.productId = productId; this.oldStock = oldStock;
        this.newStock = newStock; this.occurredAt = LocalDateTime.now();
    }
    // getters
    public Long getProductId() { return productId; }
    public int getOldStock() { return oldStock; }
    public int getNewStock() { return newStock; }
}

// 库存服务（主题）
@Service
public class InventoryService {
    private final EventPublisher<StockUpdatedEvent> publisher = new EventPublisher<>();

    // 观察者订阅
    public void addStockListener(EventListener<StockUpdatedEvent> listener) {
        publisher.subscribe(listener);
    }

    public void updateStock(Long productId, int newStock) {
        int oldStock = getCurrentStock(productId);
        saveStock(productId, newStock); // 更新库存
        publisher.publish(new StockUpdatedEvent(productId, oldStock, newStock)); // 发布事件
    }

    private int getCurrentStock(Long productId) { return 100; }
    private void saveStock(Long productId, int stock) { /* 写数据库 */ }
}

// 多个观察者分别订阅
// 低库存预警
InventoryAlertListener alertListener = new InventoryAlertListener();
inventoryService.addStockListener(event -> {
    if (event.getNewStock() < 10) {
        System.out.println("⚠️ 库存预警！商品 " + event.getProductId() + " 库存仅剩 " + event.getNewStock());
    }
});

// 搜索索引更新
inventoryService.addStockListener(event -> {
    System.out.println("更新搜索索引：商品 " + event.getProductId() + " 库存 " + event.getNewStock());
});
```

## 三、Spring ApplicationEvent 实战

```java
// 定义领域事件
public class OrderPaidEvent extends ApplicationEvent {
    private final Order order;
    public OrderPaidEvent(Object source, Order order) { super(source); this.order = order; }
    public Order getOrder() { return order; }
}

// 监听器：库存扣减（同步）
@Component
@Order(1)
public class InventoryDeductListener {
    @EventListener
    @Transactional(propagation = Propagation.REQUIRES_NEW)
    public void handleOrderPaid(OrderPaidEvent event) {
        event.getOrder().getItems().forEach(item ->
            inventoryService.deduct(item.getProductId(), item.getQuantity()));
    }
}

// 监听器：积分赠送（异步）
@Component
@Order(2)
public class PointsGrantListener {
    @Async
    @EventListener
    public void handleOrderPaid(OrderPaidEvent event) {
        Order order = event.getOrder();
        int points = order.getTotalAmount().intValue();
        pointsService.grant(order.getUserId(), points);
    }
}

// 监听器：发货通知（异步）
@Component
@Order(3)
public class ShippingNotifyListener {
    @Async
    @EventListener
    public void handleOrderPaid(OrderPaidEvent event) {
        notificationService.sendPaymentConfirmation(event.getOrder());
    }
}

// 发布事件
@Service
public class OrderService {
    @Autowired private ApplicationEventPublisher eventPublisher;

    @Transactional
    public void processPayment(Long orderId) {
        Order order = orderRepository.findById(orderId).orElseThrow();
        order.markAsPaid();
        orderRepository.save(order);
        eventPublisher.publishEvent(new OrderPaidEvent(this, order)); // 触发所有监听器
    }
}
```
