# 命令模式（Command）

> 将请求封装成对象，支持**撤销/重做、排队、延迟执行**。命令对象包含执行者（Receiver）和操作信息，调用者（Invoker）持有命令对象并在合适时机触发。

---

## 故事引入

### 《把意大利炮给我拿来！》

李云龙攻打平安县城时，大喊："二营长，你他娘的意大利炮呢？给我拉上来！"

通信兵不关心意大利炮在哪、怎么拉，他只需要传达"把意大利炮拿上来"这个**命令**，炮兵班长自然会去执行。

**命令模式**：将请求封装成命令对象，使发出请求的责任和执行请求的责任分割开。

## 一、定义与意图

将请求封装成对象，支持**撤销/重做、排队、延迟执行**。命令对象包含执行者（Receiver）和操作信息，调用者（Invoker）持有命令对象并在合适时机触发。

## 二、Demo

```java
// 场景：智能家居控制台（支持撤销）
public interface SmartHomeCommand {
    void execute();
    void undo();
    String describe();
}

// 接收者：灯光控制器
public class LightController {
    private boolean on = false;
    private int brightness = 100;
    private final String room;

    public LightController(String room) { this.room = room; }

    public void turnOn(int brightness) {
        this.on = true; this.brightness = brightness;
        System.out.printf("[%s] 灯光开启，亮度 %d%%%n", room, brightness);
    }

    public void turnOff() {
        this.on = false;
        System.out.printf("[%s] 灯光关闭%n", room);
    }

    public boolean isOn() { return on; }
    public int getBrightness() { return brightness; }
}

// 具体命令：开灯
public class TurnOnLightCommand implements SmartHomeCommand {
    private final LightController light;
    private final int brightness;
    private boolean wasOn;
    private int prevBrightness;

    public TurnOnLightCommand(LightController light, int brightness) {
        this.light = light; this.brightness = brightness;
    }

    @Override
    public void execute() {
        // 保存状态以便撤销
        wasOn = light.isOn();
        prevBrightness = light.getBrightness();
        light.turnOn(brightness);
    }

    @Override
    public void undo() {
        if (wasOn) {
            light.turnOn(prevBrightness); // 恢复之前的状态
        } else {
            light.turnOff();
        }
        System.out.println("撤销：恢复灯光状态");
    }

    @Override
    public String describe() { return "开灯（亮度" + brightness + "%）"; }
}

// 宏命令：一键回家（批量执行）
public class ComeHomeCommand implements SmartHomeCommand {
    private final List<SmartHomeCommand> commands;

    public ComeHomeCommand(List<SmartHomeCommand> commands) {
        this.commands = new ArrayList<>(commands);
    }

    @Override
    public void execute() {
        System.out.println("=== 一键回家模式 ===");
        commands.forEach(SmartHomeCommand::execute);
    }

    @Override
    public void undo() {
        // 逆序撤销
        List<SmartHomeCommand> reversed = new ArrayList<>(commands);
        Collections.reverse(reversed);
        reversed.forEach(SmartHomeCommand::undo);
    }

    @Override
    public String describe() { return "一键回家（" + commands.size() + "个动作）"; }
}

// 调用者：遥控器（持有历史命令，支持多级撤销）
public class SmartRemoteControl {
    private final Deque<SmartHomeCommand> history = new ArrayDeque<>();
    private final int maxHistory;

    public SmartRemoteControl(int maxHistory) { this.maxHistory = maxHistory; }

    public void execute(SmartHomeCommand command) {
        command.execute();
        history.push(command);
        if (history.size() > maxHistory) history.pollLast(); // 超出历史上限
        System.out.println("已执行: " + command.describe());
    }

    public void undo() {
        if (history.isEmpty()) {
            System.out.println("没有可撤销的操作");
            return;
        }
        SmartHomeCommand last = history.pop();
        last.undo();
        System.out.println("已撤销: " + last.describe());
    }
}
```
