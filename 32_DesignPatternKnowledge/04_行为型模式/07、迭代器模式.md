# 迭代器模式（Iterator）

> 提供一种方法**顺序访问集合中的元素**，而不暴露集合的内部结构。Java 集合框架的 `Iterator` 接口是这个模式的标准实现。

## 故事引入

### 《小店儿菜单的故事》

一家小店的菜单是手写的，上面密密麻麻写了几百道菜。顾客想找某道菜，只能从头一行行看，非常麻烦。

后来老板把菜单分类整理：凉菜、热菜、主食、饮品……每类用不同页面，顾客想找什么直接翻到对应分类，一目了然。

**迭代器模式**：提供一种方法顺序访问集合中的各个元素，而不暴露其内部表示。

## 一、定义与意图

## 二、Demo

```java
// 场景：分页查询迭代器（懒加载，每次只拉一页数据）
public class PagedIterator<T> implements Iterator<T> {
    private final Function<PageRequest, Page<T>> dataFetcher;
    private final int pageSize;
    private int currentPage = 0;
    private List<T> currentPageData = new ArrayList<>();
    private int indexInPage = 0;
    private boolean hasMorePages = true;

    public PagedIterator(Function<PageRequest, Page<T>> dataFetcher, int pageSize) {
        this.dataFetcher = dataFetcher;
        this.pageSize = pageSize;
        fetchNextPage(); // 预加载第一页
    }

    @Override
    public boolean hasNext() {
        if (indexInPage < currentPageData.size()) return true;
        if (!hasMorePages) return false;
        fetchNextPage();
        return !currentPageData.isEmpty();
    }

    @Override
    public T next() {
        if (!hasNext()) throw new NoSuchElementException();
        return currentPageData.get(indexInPage++);
    }

    private void fetchNextPage() {
        Page<T> page = dataFetcher.apply(PageRequest.of(currentPage, pageSize));
        currentPageData = page.getContent();
        indexInPage = 0;
        currentPage++;
        hasMorePages = page.hasNext();
        System.out.println("加载第 " + currentPage + " 页，共 " + currentPageData.size() + " 条");
    }
}

// 使用：对调用方完全透明，感知不到分页
PagedIterator<User> iterator = new PagedIterator<>(
    pageReq -> userRepository.findAll(pageReq), 100
);

while (iterator.hasNext()) {
    User user = iterator.next();
    processUser(user); // 处理每个用户，不用关心背后的分页逻辑
}

// 包装成 Iterable，支持 for-each
public class PagedIterable<T> implements Iterable<T> {
    private final Function<PageRequest, Page<T>> fetcher;
    private final int pageSize;

    public PagedIterable(Function<PageRequest, Page<T>> fetcher, int pageSize) {
        this.fetcher = fetcher; this.pageSize = pageSize;
    }

    @Override
    public Iterator<T> iterator() { return new PagedIterator<>(fetcher, pageSize); }
}

for (User user : new PagedIterable<>(userRepository::findAll, 100)) {
    processUser(user);
}
```
