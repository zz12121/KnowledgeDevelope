# 模板方法模式（Template Method）

> 在**父类（抽象类）**中定义算法骨架，把某些步骤的实现推迟到子类。模板方法用 `final` 防止子类改变流程，子类只能覆盖指定的"可变步骤"。

---

## 故事引入

### 《怀念小时候玩的红白机吗？》

小时候玩的红白机（FC），不同游戏有不同的玩法，但**开机的流程是一样的**：插卡带 → 开机 → 等待游戏加载 → 开始玩。

**模板方法**就像这个开机流程：定义一个算法的骨架，把某些步骤延迟到子类中。开机这个"模板"是固定的，但玩什么游戏（具体实现）由你决定。

## 二、Demo

```java
// 数据导入模板：固定"读文件→解析→验证→存库"流程，解析方式可变
public abstract class DataImporter<T> {

    // 模板方法：固定流程骨架
    public final ImportResult importData(String filePath) {
        List<String> rawLines = readFile(filePath);          // 1. 读文件（固定）
        if (rawLines.isEmpty()) return ImportResult.empty();

        List<T> records = parseLines(rawLines);              // 2. 解析（可变）
        List<ValidationError> errors = validateRecords(records); // 3. 验证（可变）
        if (!errors.isEmpty()) {
            if (!shouldContinueOnError()) {                  // 4. 钩子：是否遇错继续
                return ImportResult.fail(errors);
            }
        }
        int savedCount = saveRecords(records);               // 5. 存库（可变）
        afterImport(savedCount);                             // 6. 钩子：导入后处理

        return ImportResult.success(savedCount, errors);
    }

    // 固定步骤
    private List<String> readFile(String filePath) {
        try {
            return Files.readAllLines(Paths.get(filePath), StandardCharsets.UTF_8);
        } catch (Exception e) { throw new RuntimeException("读文件失败: " + filePath, e); }
    }

    // 抽象步骤（子类必须实现）
    protected abstract List<T> parseLines(List<String> lines);
    protected abstract List<ValidationError> validateRecords(List<T> records);
    protected abstract int saveRecords(List<T> records);

    // 钩子方法（子类可选覆盖）
    protected boolean shouldContinueOnError() { return false; }  // 默认遇错停止
    protected void afterImport(int count) {
        System.out.println("导入完成，共导入 " + count + " 条记录");
    }
}

// 具体实现：CSV 用户导入
@Component
public class UserCsvImporter extends DataImporter<User> {
    @Autowired private UserRepository userRepository;
    @Autowired private UserValidator userValidator;

    @Override
    protected List<User> parseLines(List<String> lines) {
        return lines.stream()
                .filter(line -> !line.startsWith("#")) // 跳过注释行
                .map(line -> {
                    String[] parts = line.split(",");
                    return User.builder()
                            .username(parts[0].trim())
                            .email(parts[1].trim())
                            .phone(parts.length > 2 ? parts[2].trim() : null)
                            .build();
                })
                .collect(Collectors.toList());
    }

    @Override
    protected List<ValidationError> validateRecords(List<User> users) {
        return users.stream()
                .flatMap(u -> userValidator.validate(u).getErrors().stream())
                .collect(Collectors.toList());
    }

    @Override
    protected int saveRecords(List<User> users) {
        userRepository.saveAll(users);
        return users.size();
    }

    @Override
    protected boolean shouldContinueOnError() { return true; } // 用户导入允许部分失败
}
```
