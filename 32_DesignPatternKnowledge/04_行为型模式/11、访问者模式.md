# 访问者模式（Visitor）

> 把**数据结构**和**作用于其上的操作**分离。当数据结构稳定但需要频繁新增操作时，新增访问者类即可，不修改数据结构。

---

## 故事引入

### 《小王与他的Apple商店》

Apple商店对不同人群有不同优惠政策：
- 学生消费者：教育优惠
- 普通消费者：原价
- 企业大客户：批量采购折扣

如果用if-else判断用户类型，代码会越来越乱。但如果我们把"优惠政策"当作一个**访问者**，让不同用户类型"接待"这个访问者，就清晰多了。

**访问者模式**：表示一个作用于某个对象结构中的各元素的操作，使你可以在不改变元素类的前提下定义新操作。

---

## 一、定义与意图

把**数据结构**和**作用于其上的操作**分离。当数据结构稳定但需要频繁新增操作时，新增访问者类即可，不修改数据结构。

## 二、Demo

```java
// 场景：AST（抽象语法树）统计分析
// 节点类型（稳定，不经常变）
public interface AstNode { void accept(AstVisitor visitor); }

public class NumberNode implements AstNode {
    public final double value;
    public NumberNode(double value) { this.value = value; }
    public void accept(AstVisitor visitor) { visitor.visit(this); }
}

public class VariableNode implements AstNode {
    public final String name;
    public VariableNode(String name) { this.name = name; }
    public void accept(AstVisitor visitor) { visitor.visit(this); }
}

public class BinaryOpNode implements AstNode {
    public final String operator;
    public final AstNode left, right;
    public BinaryOpNode(String op, AstNode left, AstNode right) {
        this.operator = op; this.left = left; this.right = right;
    }
    public void accept(AstVisitor visitor) { visitor.visit(this); }
}

// 访问者接口（新增操作 = 新增实现类，不改节点）
public interface AstVisitor {
    void visit(NumberNode node);
    void visit(VariableNode node);
    void visit(BinaryOpNode node);
}

// 访问者1：计算表达式的值
public class EvaluateVisitor implements AstVisitor {
    private final Map<String, Double> variables;
    private double result;

    public EvaluateVisitor(Map<String, Double> variables) { this.variables = variables; }

    public void visit(NumberNode node) { result = node.value; }

    public void visit(VariableNode node) {
        result = variables.getOrDefault(node.name, 0.0);
    }

    public void visit(BinaryOpNode node) {
        node.left.accept(this); double left = result;
        node.right.accept(this); double right = result;
        result = switch (node.operator) {
            case "+" -> left + right;
            case "-" -> left - right;
            case "*" -> left * right;
            case "/" -> right != 0 ? left / right : Double.NaN;
            default -> throw new UnsupportedOperationException();
        };
    }

    public double getResult() { return result; }
}

// 访问者2：统计变量使用次数（新增操作，节点类不变）
public class VariableCountVisitor implements AstVisitor {
    private final Map<String, Integer> counts = new HashMap<>();

    public void visit(NumberNode node) {} // 数字节点不计入
    public void visit(VariableNode node) { counts.merge(node.name, 1, Integer::sum); }
    public void visit(BinaryOpNode node) { node.left.accept(this); node.right.accept(this); }

    public Map<String, Integer> getCounts() { return Collections.unmodifiableMap(counts); }
}

// 使用：(x + 2) * (y - x)
AstNode ast = new BinaryOpNode("*",
    new BinaryOpNode("+", new VariableNode("x"), new NumberNode(2)),
    new BinaryOpNode("-", new VariableNode("y"), new VariableNode("x"))
);

EvaluateVisitor evaluator = new EvaluateVisitor(Map.of("x", 3.0, "y", 7.0));
ast.accept(evaluator);
System.out.println("结果: " + evaluator.getResult()); // (3+2)*(7-3) = 20.0

VariableCountVisitor counter = new VariableCountVisitor();
ast.accept(counter);
System.out.println("变量使用: " + counter.getCounts()); // {x=2, y=1}
```
