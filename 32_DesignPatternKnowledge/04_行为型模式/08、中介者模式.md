# 中介者模式（Mediator）

> 用中介者对象来封装一系列对象之间的交互，使各对象不再直接相互引用，将**网状耦合**简化为**星形结构**。

---

## 故事引入

### 《为什么要找房子要找中介？》

租房市场上有房东和租客：
- 房东A有房子出租，想找租客
- 租客B想租房，不知道在哪找房东
- 房东C、D、E……也都有房子
- 租客F、G、H……也都在找房

如果让房东和租客直接对接，会非常混乱。于是有了**中介**：房东把房子挂到中介，租客从中介找房，中介负责匹配。

**中介者模式**：用一个中介对象来简化对象之间的交互关系。

---

## 一、定义与意图

用中介者对象来封装一系列对象之间的交互，使各对象不再直接相互引用，将**网状耦合**简化为**星形结构**。

---

## 二、Demo

```java
// 场景：机场起降协调系统
public interface AirTrafficControl {
    void requestLanding(Airplane airplane);
    void requestTakeoff(Airplane airplane);
    void notifyAll(String message);
}

// 具体中介者：塔台
public class ControlTower implements AirTrafficControl {
    private final List<Airplane> registeredPlanes = new ArrayList<>();
    private final Set<String> landingApprovals = new ConcurrentHashSet<>();

    public void register(Airplane airplane) { registeredPlanes.add(airplane); }

    @Override
    public void requestLanding(Airplane airplane) {
        boolean runwayClear = isRunwayClear();
        if (runwayClear) {
            landingApprovals.add(airplane.getFlightId());
            System.out.printf("塔台 -> %s: 批准降落，跑道01可用%n", airplane.getFlightId());
            airplane.receiveLandingApproval(true);
        } else {
            System.out.printf("塔台 -> %s: 等待，跑道繁忙，进入等待航线%n", airplane.getFlightId());
            airplane.receiveLandingApproval(false);
        }
    }

    @Override
    public void requestTakeoff(Airplane airplane) {
        System.out.printf("塔台 -> %s: 可以起飞，风速正常%n", airplane.getFlightId());
        // 通知其他飞机跑道被占用
        notifyOthers(airplane, "跑道" + airplane.getFlightId() + "正在起飞，请注意间距");
        airplane.receiveTakeoffApproval();
    }

    @Override
    public void notifyAll(String message) {
        registeredPlanes.forEach(p -> p.receiveMessage(message));
    }

    private void notifyOthers(Airplane except, String message) {
        registeredPlanes.stream()
                .filter(p -> !p.equals(except))
                .forEach(p -> p.receiveMessage(message));
    }

    private boolean isRunwayClear() { return landingApprovals.isEmpty(); }
}

// 同事类：飞机（只和塔台通信，不直接和其他飞机通信）
public class Airplane {
    private final String flightId;
    private final AirTrafficControl tower; // 引用中介者

    public Airplane(String flightId, AirTrafficControl tower) {
        this.flightId = flightId; this.tower = tower;
    }

    public void requestLanding() {
        System.out.printf("%s: 请求降落%n", flightId);
        tower.requestLanding(this); // 通过中介者请求，不直接联系其他飞机
    }

    public void requestTakeoff() {
        System.out.printf("%s: 请求起飞%n", flightId);
        tower.requestTakeoff(this);
    }

    public void receiveLandingApproval(boolean approved) {
        System.out.printf("%s: 收到降落%s%n", flightId, approved ? "批准" : "拒绝");
    }

    public void receiveTakeoffApproval() {
        System.out.printf("%s: 收到起飞批准，正在起飞...%n", flightId);
    }

    public void receiveMessage(String message) {
        System.out.printf("%s 收到通知: %s%n", flightId, message);
    }

    public String getFlightId() { return flightId; }
}
```
