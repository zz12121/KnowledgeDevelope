# 组合模式（Composite）

> 将对象组合成**树形结构**来表示"部分-整体"层次，使客户端对单个对象和组合对象的操作方式保持一致。适合处理树形结构数据。

---

## 故事引入

### 《庞大的组织架构带来的烦恼》

某大型集团有复杂的组织架构：
- 集团公司
  - 华东区
    - 上海分公司
      - 销售部
      - 技术部
    - 杭州分公司
      - 销售部
      - 技术部
  - 华南区
    - ……
    
每次统计人数或计算薪资，都需要区分是"单个员工"还是"整个部门"，代码很难写。

后来采用了一套系统：**员工和部门都实现同一个"组织节点"接口**，员工没有下属，部门有下属。统计的时候，无论给的是员工还是部门，代码都能统一处理。

**组合模式**：让单个对象和组合对象具有一致的访问性。

---

## 一、定义与意图

将对象组合成**树形结构**来表示"部分-整体"层次，使客户端对单个对象和组合对象的操作方式保持一致。适合处理树形结构数据。

---

## 二、类图角色

```
Component（抽象组件）
├── Leaf（叶子，无子节点）
└── Composite（容器，包含子 Component 集合）
```

## 三、Demo

```java
// 场景：公司组织架构（部门/员工树）
public abstract class OrgNode {
    protected String name;
    protected String position;

    public OrgNode(String name, String position) {
        this.name = name; this.position = position;
    }

    public abstract double getTotalSalary();
    public abstract int getHeadcount();
    public abstract void display(int depth);

    // 默认不支持子节点操作
    public void add(OrgNode node) { throw new UnsupportedOperationException(); }
    public void remove(OrgNode node) { throw new UnsupportedOperationException(); }
    public List<OrgNode> getChildren() { return List.of(); }
}

// 叶子节点：员工
public class Employee extends OrgNode {
    private double salary;

    public Employee(String name, String position, double salary) {
        super(name, position);
        this.salary = salary;
    }

    @Override
    public double getTotalSalary() { return salary; }

    @Override
    public int getHeadcount() { return 1; }

    @Override
    public void display(int depth) {
        System.out.printf("%s👤 %s（%s）- ¥%.0f%n",
                "  ".repeat(depth), name, position, salary);
    }
}

// 容器节点：部门
public class Department extends OrgNode {
    private final List<OrgNode> members = new ArrayList<>();

    public Department(String name) { super(name, "部门"); }

    @Override
    public void add(OrgNode node) { members.add(node); }

    @Override
    public void remove(OrgNode node) { members.remove(node); }

    @Override
    public List<OrgNode> getChildren() { return Collections.unmodifiableList(members); }

    @Override
    public double getTotalSalary() {
        return members.stream().mapToDouble(OrgNode::getTotalSalary).sum(); // 递归汇总
    }

    @Override
    public int getHeadcount() {
        return members.stream().mapToInt(OrgNode::getHeadcount).sum(); // 递归计数
    }

    @Override
    public void display(int depth) {
        System.out.printf("%s📁 %s（共%d人，薪酬¥%.0f）%n",
                "  ".repeat(depth), name, getHeadcount(), getTotalSalary());
        members.forEach(m -> m.display(depth + 1));
    }
}

// 使用
Department company = new Department("科技公司");
Department techDept = new Department("技术部");
Department bizDept = new Department("业务部");

techDept.add(new Employee("张三", "架构师", 35000));
techDept.add(new Employee("李四", "高级工程师", 28000));
techDept.add(new Employee("王五", "工程师", 18000));

Department frontendTeam = new Department("前端组");
frontendTeam.add(new Employee("赵六", "前端工程师", 20000));
frontendTeam.add(new Employee("钱七", "前端工程师", 18000));
techDept.add(frontendTeam);

bizDept.add(new Employee("孙八", "产品经理", 25000));
bizDept.add(new Employee("周九", "运营", 15000));

company.add(techDept);
company.add(bizDept);

company.display(0);
System.out.printf("公司总人数: %d，总薪酬: ¥%.0f%n", company.getHeadcount(), company.getTotalSalary());
```
