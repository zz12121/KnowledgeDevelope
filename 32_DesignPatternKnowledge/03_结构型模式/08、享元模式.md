# 享元模式（Flyweight）

> 通过**共享**来有效支持大量细粒度对象。把对象状态分成**内在状态**（不变可共享）和**外在状态**（变化不共享），只共享内在状态，外在状态由客户端传入。

---

## 故事引入

### 《开发一个疫苗管理平台系统》

某市要开发一个核酸检测管理平台，需要采集2000万人的个人信息，包括：姓名、居住地址、检测医院、检测品牌等。

仔细分析发现：姓名和地址是每个人不同的，但检测医院和检测品牌是固定的（全市只有10家医院、5种检测品牌）。

如果每个人都存储完整的医院和品牌信息，空间浪费巨大！更好的办法是：**把这些固定信息池化，大家共享**。

**享元模式**：运用共享技术有效支持大量细粒度对象的复用。

---

## 一、定义与意图

通过**共享**来有效支持大量细粒度对象。把对象状态分成**内在状态**（不变可共享）和**外在状态**（变化不共享），只共享内在状态，外在状态由客户端传入。

## 三、Demo

```java
// 场景：在线文档编辑器，每个字符对象共享字体/大小/颜色
// 享元对象（内在状态：字体属性，多字符共享）
public class CharacterStyle {
    private final String fontFamily;
    private final int fontSize;
    private final String color;
    private final boolean bold;
    private final boolean italic;

    public CharacterStyle(String fontFamily, int fontSize, String color, boolean bold, boolean italic) {
        this.fontFamily = fontFamily; this.fontSize = fontSize;
        this.color = color; this.bold = bold; this.italic = italic;
    }

    // 渲染时，外在状态（位置/字符本身）从外部传入
    public void render(char character, int row, int col) {
        System.out.printf("渲染字符 '%c' 在(%d,%d)，字体=%s，大小=%d，颜色=%s，粗体=%s%n",
                character, row, col, fontFamily, fontSize, color, bold);
    }

    @Override
    public String toString() {
        return String.format("%s-%d-%s-%b-%b", fontFamily, fontSize, color, bold, italic);
    }
}

// 享元工厂
public class CharacterStyleFactory {
    private static final Map<String, CharacterStyle> pool = new HashMap<>();
    private static int createCount = 0;

    public static CharacterStyle getStyle(String fontFamily, int fontSize, String color,
                                          boolean bold, boolean italic) {
        String key = String.format("%s-%d-%s-%b-%b", fontFamily, fontSize, color, bold, italic);
        return pool.computeIfAbsent(key, k -> {
            createCount++;
            System.out.println("创建新样式对象: " + key);
            return new CharacterStyle(fontFamily, fontSize, color, bold, italic);
        });
    }

    public static int getPoolSize() { return pool.size(); }
    public static int getCreateCount() { return createCount; }
}

// 字符对象（只存储外在状态：字符本身）
public class DocumentCharacter {
    private final char ch;           // 外在状态：具体字符
    private final int row;           // 外在状态：行
    private final int col;           // 外在状态：列
    private final CharacterStyle style; // 内在状态：共享的样式对象

    public DocumentCharacter(char ch, int row, int col, CharacterStyle style) {
        this.ch = ch; this.row = row; this.col = col; this.style = style;
    }

    public void render() { style.render(ch, row, col); }
}

// 使用：10000个字符，但只有少量样式对象
public class Document {
    public static void main(String[] args) {
        List<DocumentCharacter> chars = new ArrayList<>();
        String text = "Hello World! This is a sample document with many characters.";
        String[] fonts = {"宋体", "微软雅黑", "楷体"};
        int[] sizes = {12, 14, 16};
        String[] colors = {"black", "red", "blue"};

        // 创建大量字符，但样式对象复用
        for (int i = 0; i < 10000; i++) {
            char c = text.charAt(i % text.length());
            CharacterStyle style = CharacterStyleFactory.getStyle(
                fonts[i % 3], sizes[i % 3], colors[i % 3], i % 5 == 0, i % 7 == 0
            );
            chars.add(new DocumentCharacter(c, i / 80, i % 80, style));
        }

        System.out.printf("字符对象数: %d，样式对象数: %d（节省内存约%.1f%%）%n",
                chars.size(), CharacterStyleFactory.getPoolSize(),
                (1 - (double) CharacterStyleFactory.getPoolSize() / chars.size()) * 100);
    }
}
```

## 四、实际业务场景

```java
// 场景：Spring 中的 Bean 本质上就是享元模式
// 每个单例 Bean 都被工厂创建一次，然后在整个应用中被复用（共享）
// DefaultSingletonBeanRegistry 中的 singletonObjects 就是享元池

// 自定义享元：地理位置省市数据
@Component
public class RegionFlyweightFactory {
    // 省市数据是内在状态（不变），加载一次后缓存复用
    private final Map<String, Region> regionCache = new ConcurrentHashMap<>();

    @PostConstruct
    public void init() {
        // 一次性加载所有地区数据到缓存
        loadAllRegions().forEach(r -> regionCache.put(r.getCode(), r));
    }

    public Region getRegion(String code) { return regionCache.get(code); }

    private List<Region> loadAllRegions() { /* 从数据库加载 */ return List.of(); }
}
```
