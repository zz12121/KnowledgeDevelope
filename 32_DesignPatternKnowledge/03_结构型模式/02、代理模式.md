# 代理模式（Proxy）

> 为其他对象提供一种代理来**控制对该对象的访问**。代理对象与目标对象实现相同的接口，客户端通过代理间接访问真实对象，可以在访问前后添加额外逻辑。

---

## 故事引入

### 《离婚，为什么需要离婚律师？》

小李和妻子感情破裂，决定离婚。但离婚涉及财产分割、孩子抚养权、债务处理等复杂问题。小李自己去法院办理？肯定不行——他不懂法律！

于是小李请了一个**律师**。律师帮他处理所有法律事务，小李只需要配合签字就行。

**代理模式**正是如此：不直接访问真实对象，而是通过代理对象来间接访问，代理可以在访问前后添加额外逻辑（律师帮你打官司）。

## 一、定义与意图

为其他对象提供一种代理来**控制对该对象的访问**。代理对象与目标对象实现相同的接口，客户端通过代理间接访问真实对象，可以在访问前后添加额外逻辑。

---

## 二、类图角色

```
Subject（接口）
├── RealSubject（真实对象）
└── Proxy（代理对象）
    ├── - realSubject: RealSubject
    └── + request()  // 在调用 realSubject.request() 前后加逻辑
```

## 三、三种代理类型

### 静态代理

```java
// 业务接口
public interface OrderService {
    Order createOrder(CreateOrderRequest request);
    Order findById(Long id);
}

// 真实对象
@Service
public class OrderServiceImpl implements OrderService {
    @Autowired private OrderRepository orderRepository;

    public Order createOrder(CreateOrderRequest request) {
        Order order = Order.from(request);
        orderRepository.save(order);
        return order;
    }

    public Order findById(Long id) { return orderRepository.findById(id).orElseThrow(); }
}

// 静态代理：手动编写，为 OrderService 添加日志
public class OrderServiceLoggingProxy implements OrderService {
    private final OrderService target; // 被代理的真实对象

    public OrderServiceLoggingProxy(OrderService target) {
        this.target = target;
    }

    @Override
    public Order createOrder(CreateOrderRequest request) {
        long start = System.currentTimeMillis();
        System.out.println("开始创建订单，用户ID: " + request.getUserId());
        try {
            Order order = target.createOrder(request);
            System.out.printf("订单创建成功，ID=%d，耗时%dms%n",
                    order.getId(), System.currentTimeMillis() - start);
            return order;
        } catch (Exception e) {
            System.out.println("订单创建失败: " + e.getMessage());
            throw e;
        }
    }

    @Override
    public Order findById(Long id) {
        System.out.println("查询订单: " + id);
        return target.findById(id);
    }
}
```

### JDK 动态代理（基于接口）

```java
// 通用日志 InvocationHandler（可以代理任何接口）
public class LoggingInvocationHandler implements InvocationHandler {
    private final Object target;
    private final String serviceName;

    public LoggingInvocationHandler(Object target) {
        this.target = target;
        this.serviceName = target.getClass().getSimpleName();
    }

    @Override
    public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
        long start = System.nanoTime();
        System.out.printf("[%s.%s] 开始执行%n", serviceName, method.getName());
        try {
            Object result = method.invoke(target, args);
            System.out.printf("[%s.%s] 执行成功，耗时%.2fms%n",
                    serviceName, method.getName(),
                    (System.nanoTime() - start) / 1_000_000.0);
            return result;
        } catch (InvocationTargetException e) {
            System.out.printf("[%s.%s] 执行异常: %s%n",
                    serviceName, method.getName(), e.getCause().getMessage());
            throw e.getCause();
        }
    }
}

// 代理工厂工具类
public class ProxyFactory {
    @SuppressWarnings("unchecked")
    public static <T> T createLoggingProxy(T target, Class<T> interfaceType) {
        return (T) Proxy.newProxyInstance(
            target.getClass().getClassLoader(),
            new Class[]{interfaceType},
            new LoggingInvocationHandler(target)
        );
    }
}

// 使用
OrderService orderService = ProxyFactory.createLoggingProxy(
    new OrderServiceImpl(), OrderService.class
);
orderService.findById(1L); // 自动打印日志
```

### CGLIB 代理（基于继承，无需接口）

```java
// 假设 ProductService 没有接口
public class ProductService {
    public Product findById(Long id) {
        System.out.println("查询商品: " + id);
        return new Product(id, "商品" + id);
    }

    public void updateStock(Long productId, int delta) {
        System.out.println("更新库存: product=" + productId + ", delta=" + delta);
    }
}

// CGLIB 代理（Spring 项目可直接使用，Spring Core 包含 CGLIB）
public class CglibLoggingProxy implements MethodInterceptor {
    @Override
    public Object intercept(Object obj, Method method, Object[] args, MethodProxy proxy) throws Throwable {
        System.out.println("CGLIB 拦截: " + method.getName());
        long start = System.currentTimeMillis();
        try {
            return proxy.invokeSuper(obj, args); // 调用父类（被代理类）的方法
        } finally {
            System.out.println(method.getName() + " 耗时: " + (System.currentTimeMillis() - start) + "ms");
        }
    }
}

// 创建 CGLIB 代理
Enhancer enhancer = new Enhancer();
enhancer.setSuperclass(ProductService.class);
enhancer.setCallback(new CglibLoggingProxy());
ProductService proxy = (ProductService) enhancer.create();
proxy.findById(1L);
```

## 四、实际业务场景

```java
// 场景：用注解 + 动态代理实现缓存切面
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Cacheable {
    String key();
    int ttlSeconds() default 300;
}

public class CacheProxy implements InvocationHandler {
    private final Object target;
    private final Map<String, Object> cache = new ConcurrentHashMap<>();

    public CacheProxy(Object target) { this.target = target; }

    @Override
    public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
        Cacheable cacheable = method.getAnnotation(Cacheable.class);
        if (cacheable == null) {
            return method.invoke(target, args); // 没有注解，直接执行
        }
        String cacheKey = cacheable.key() + Arrays.toString(args);
        return cache.computeIfAbsent(cacheKey, k -> {
            try {
                System.out.println("缓存未命中，执行方法: " + method.getName());
                return method.invoke(target, args);
            } catch (Exception e) { throw new RuntimeException(e); }
        });
    }
}
```
