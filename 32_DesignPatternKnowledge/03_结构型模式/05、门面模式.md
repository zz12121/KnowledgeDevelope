# 门面模式（Facade）

> 为子系统中的一组接口提供一个统一的**简化入口**，隐藏子系统内部的复杂性，让客户端更容易使用。

---

## 故事引入

### 《办理入职怎么这么难？》

小王大学毕业，去一家公司办理入职。前台递给他一张单子，上面列了30多项需要他自己去办的事项：
- 办理档案（去上家公司）
- 办理公积金（去公积金中心）
- 领取电脑（自己去电脑店采购）
- 安装办公软件（按列表安装）
- 确定工位（自己申请）
- 办理工资卡（去银行）
- 办理工卡（去制卡处）
- ……

小王崩溃了：我就是办个入职，为什么要跑遍全城？

后来另一家公司采用了**HR一站式服务**：所有资料在会议室统一收集，HR帮忙跑腿办理，小王只需要签个字就行。

**门面模式**：为复杂子系统提供统一的简化入口。

---

## 一、定义与意图

为子系统中的一组接口提供一个统一的**简化入口**，隐藏子系统内部的复杂性，让客户端更容易使用。

---

## 二、类图角色

```
Facade（门面）
├── - subsystemA: SubsystemA
├── - subsystemB: SubsystemB
├── - subsystemC: SubsystemC
└── + simplifiedOperation()  // 封装多个子系统的调用
```

## 三、Demo

```java
// 场景：电商下单门面（协调库存、支付、物流、通知多个子系统）
@Service
public class PlaceOrderFacade {
    private final InventoryService inventoryService;
    private final OrderRepository orderRepository;
    private final PaymentService paymentService;
    private final ShippingService shippingService;
    private final NotificationService notificationService;
    private final CouponService couponService;
    private final PointsService pointsService;

    @Autowired
    public PlaceOrderFacade(InventoryService inventoryService,
                            OrderRepository orderRepository,
                            PaymentService paymentService,
                            ShippingService shippingService,
                            NotificationService notificationService,
                            CouponService couponService,
                            PointsService pointsService) {
        this.inventoryService = inventoryService;
        this.orderRepository = orderRepository;
        this.paymentService = paymentService;
        this.shippingService = shippingService;
        this.notificationService = notificationService;
        this.couponService = couponService;
        this.pointsService = pointsService;
    }

    // 门面方法：客户端只调用这一个接口，不需要了解内部有多少个子系统
    @Transactional
    public PlaceOrderResult placeOrder(PlaceOrderRequest request) {
        // 1. 校验并预扣库存
        List<InventoryResult> inventoryResults = inventoryService.batchCheckAndLock(request.getItems());
        if (inventoryResults.stream().anyMatch(r -> !r.isSuccess())) {
            return PlaceOrderResult.fail("库存不足: " + getInsufficientItems(inventoryResults));
        }

        // 2. 计算优惠
        BigDecimal discount = BigDecimal.ZERO;
        if (request.getCouponId() != null) {
            discount = couponService.calculateDiscount(request.getCouponId(), request.getTotalAmount());
        }

        // 3. 创建订单
        Order order = Order.builder()
                .userId(request.getUserId())
                .items(request.getItems())
                .totalAmount(request.getTotalAmount().subtract(discount))
                .shippingAddress(request.getShippingAddress())
                .build();
        orderRepository.save(order);

        // 4. 处理支付
        PaymentResult payResult = paymentService.processPayment(order.getId(), order.getTotalAmount(), request.getPaymentMethod());
        if (!payResult.isSuccess()) {
            inventoryService.releaseLock(order.getId()); // 支付失败，释放库存锁
            throw new PaymentException("支付失败: " + payResult.getMessage());
        }

        // 5. 创建运单
        String trackingNo = shippingService.createShipment(order.getId(), request.getShippingAddress());

        // 6. 赠送积分
        if (payResult.isSuccess()) {
            pointsService.addPoints(request.getUserId(), calcPoints(order.getTotalAmount()));
        }

        // 7. 发送通知
        notificationService.sendOrderConfirmation(request.getUserId(), order.getId(), trackingNo);

        return PlaceOrderResult.success(order.getId(), trackingNo);
    }

    private String getInsufficientItems(List<InventoryResult> results) {
        return results.stream().filter(r -> !r.isSuccess())
                .map(InventoryResult::getProductName).collect(Collectors.joining(", "));
    }

    private int calcPoints(BigDecimal amount) { return amount.intValue(); }
}

// 控制器层极其简洁
@RestController
public class OrderController {
    @Autowired private PlaceOrderFacade placeOrderFacade;

    @PostMapping("/orders")
    public ResponseEntity<PlaceOrderResult> placeOrder(@RequestBody PlaceOrderRequest request) {
        PlaceOrderResult result = placeOrderFacade.placeOrder(request);
        return ResponseEntity.ok(result);
    }
}
```
