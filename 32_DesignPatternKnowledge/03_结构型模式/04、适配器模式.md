# 适配器模式（Adapter）

> 将一个类的接口转换成客户端期望的另一个接口，让**原本接口不兼容**的类可以一起工作。解决新旧系统集成、第三方 SDK 对接时的接口不匹配问题。

---

## 故事引入

### 《买了苹果笔记本的尴尬》

小明买了最新款的苹果笔记本，外观时尚，性能强大。但当他想投影到公司会议室的老旧投影仪时傻眼了——MacBook只有Type-C接口，而投影仪是VGA接口！

这时候小明需要一个**转接头**（Adapter）——把Type-C转成VGA，这样就能连接了。

**适配器模式**就是这样的"转接头"——把不兼容的接口转换成兼容的。

---

## 一、定义与意图

将一个类的接口转换成客户端期望的另一个接口，让**原本接口不兼容**的类可以一起工作。解决新旧系统集成、第三方 SDK 对接时的接口不匹配问题。

---

## 二、类图角色

```
Target（目标接口）
Adaptee（被适配者，已有的不兼容类）
Adapter（适配器）
├── 类适配器：extends Adaptee, implements Target
└── 对象适配器：implements Target, has-a Adaptee
```

## 三、Demo

```java
// 场景：统一不同厂商的短信服务接口
// 目标接口（我们系统希望使用的统一接口）
public interface SmsSender {
    SmsResult send(String phone, String content);
    SmsResult sendBatch(List<String> phones, String content);
}

// 阿里云短信 SDK（Adaptee1 ——— 不能修改）
public class AliCloudSmsClient {
    private final String accessKeyId;
    private final String accessKeySecret;

    public AliCloudSmsClient(String accessKeyId, String accessKeySecret) {
        this.accessKeyId = accessKeyId; this.accessKeySecret = accessKeySecret;
    }

    public AliSmsResponse sendSms(String phoneNumber, String signName, String templateCode, String templateParam) {
        System.out.println("阿里云短信发送: " + phoneNumber);
        return new AliSmsResponse("OK", "发送成功");
    }
}

// 腾讯云短信 SDK（Adaptee2）
public class TencentSmsClient {
    public TencentSmsResponse sendSms(String[] phoneNumbers, String sdkAppId, String signName, String templateId, String[] templateValues) {
        System.out.println("腾讯云短信发送: " + Arrays.toString(phoneNumbers));
        return new TencentSmsResponse(0, "ok");
    }
}

// 对象适配器：阿里云实现
public class AliCloudSmsAdapter implements SmsSender {
    private final AliCloudSmsClient aliClient;
    private final String signName;
    private final String templateCode;

    public AliCloudSmsAdapter(AliCloudSmsClient aliClient, String signName, String templateCode) {
        this.aliClient = aliClient; this.signName = signName; this.templateCode = templateCode;
    }

    @Override
    public SmsResult send(String phone, String content) {
        AliSmsResponse resp = aliClient.sendSms(phone, signName, templateCode,
                "{\"content\":\"" + content + "\"}");
        return new SmsResult("OK".equals(resp.getCode()), resp.getMessage());
    }

    @Override
    public SmsResult sendBatch(List<String> phones, String content) {
        List<SmsResult> results = phones.stream()
                .map(p -> send(p, content))
                .collect(Collectors.toList());
        boolean allSuccess = results.stream().allMatch(SmsResult::isSuccess);
        return new SmsResult(allSuccess, allSuccess ? "批量发送成功" : "部分失败");
    }
}

// 对象适配器：腾讯云实现
public class TencentSmsAdapter implements SmsSender {
    private final TencentSmsClient tencentClient;
    private final String sdkAppId;
    private final String signName;
    private final String templateId;

    public TencentSmsAdapter(TencentSmsClient client, String sdkAppId, String signName, String templateId) {
        this.tencentClient = client; this.sdkAppId = sdkAppId;
        this.signName = signName; this.templateId = templateId;
    }

    @Override
    public SmsResult send(String phone, String content) {
        TencentSmsResponse resp = tencentClient.sendSms(
            new String[]{phone}, sdkAppId, signName, templateId, new String[]{content});
        return new SmsResult(resp.getCode() == 0, resp.getMessage());
    }

    @Override
    public SmsResult sendBatch(List<String> phones, String content) {
        TencentSmsResponse resp = tencentClient.sendSms(
            phones.toArray(new String[0]), sdkAppId, signName, templateId, new String[]{content});
        return new SmsResult(resp.getCode() == 0, resp.getMessage());
    }
}

// 业务层只依赖统一接口，切换供应商只需换适配器
@Service
public class NotificationService {
    @Autowired private SmsSender smsSender; // 注入哪个适配器由配置决定

    public void sendVerifyCode(String phone, String code) {
        SmsResult result = smsSender.send(phone, "您的验证码是：" + code + "，5分钟内有效");
        if (!result.isSuccess()) {
            throw new BusinessException("短信发送失败: " + result.getMessage());
        }
    }
}
```
