# 抽象工厂模式（Abstract Factory）

> 提供一个创建**一系列相关或相互依赖对象**的接口，而无需指定它们的具体类。解决产品族的一致性问题——一个抽象工厂创建出来的所有产品，保证是同一个产品家族，互相兼容。

---

## 故事引入

### 《没有中间商赚差价？》（续）

从前有个 Pizza 店，老板娘直接面对顾客。后来镇上开了连锁店，每家店都要配置会做各种Pizza的师傅，成本太高。

于是总部制定了一套标准：每家店只要会"按照总部配方做Pizza"就行——这就是**抽象工厂**。

---

## 一、定义与意图

提供一个创建**一系列相关或相互依赖对象**的接口，而无需指定它们的具体类。解决产品族的一致性问题——一个抽象工厂创建出来的所有产品，保证是同一个产品家族，互相兼容。

---

## 二、类图角色

```
AbstractFactory
├── + createProductA(): AbstractProductA
└── + createProductB(): AbstractProductB

ConcreteFactory1 extends AbstractFactory  // 产品族1
ConcreteFactory2 extends AbstractFactory  // 产品族2

AbstractProductA
├── ConcreteProductA1
└── ConcreteProductA2

AbstractProductB
├── ConcreteProductB1
└── ConcreteProductB2
```

## 三、Demo

```java
// 数据访问层的抽象工厂（不同数据库实现）
public interface UserDao {
    User findById(Long id);
    void save(User user);
}

public interface OrderDao {
    Order findById(Long id);
    List<Order> findByUserId(Long userId);
    void save(Order order);
}

public interface ProductDao {
    Product findById(Long id);
    List<Product> findAll(int page, int size);
}

// 抽象工厂：创建一套完整的 DAO 产品族
public interface DaoFactory {
    UserDao createUserDao();
    OrderDao createOrderDao();
    ProductDao createProductDao();
}

// MySQL 实现族
public class MySQLUserDao implements UserDao {
    private final JdbcTemplate jdbcTemplate;
    public MySQLUserDao(JdbcTemplate jdbcTemplate) { this.jdbcTemplate = jdbcTemplate; }

    public User findById(Long id) {
        return jdbcTemplate.queryForObject("SELECT * FROM users WHERE id=?", new UserRowMapper(), id);
    }
    public void save(User user) { jdbcTemplate.update("INSERT INTO users ...", user.getUsername()); }
}

// 具体工厂：MySQL 完整实现族
public class MySQLDaoFactory implements DaoFactory {
    private final JdbcTemplate jdbcTemplate;
    public MySQLDaoFactory(JdbcTemplate jdbcTemplate) { this.jdbcTemplate = jdbcTemplate; }

    public UserDao createUserDao()       { return new MySQLUserDao(jdbcTemplate); }
    public OrderDao createOrderDao()     { return new MySQLOrderDao(jdbcTemplate); }
    public ProductDao createProductDao() { return new MySQLProductDao(jdbcTemplate); }
}

// 具体工厂：MongoDB 完整实现族（切换数据库只需换工厂）
public class MongoDaoFactory implements DaoFactory {
    private final MongoTemplate mongoTemplate;
    public MongoDaoFactory(MongoTemplate mongoTemplate) { this.mongoTemplate = mongoTemplate; }

    public UserDao createUserDao()       { return new MongoUserDao(mongoTemplate); }
    public OrderDao createOrderDao()     { return new MongoOrderDao(mongoTemplate); }
    public ProductDao createProductDao() { return new MongoProductDao(mongoTemplate); }
}

// 应用层依赖抽象工厂，不关心底层数据库
@Service
public class UserService {
    private final UserDao userDao;

    @Autowired
    public UserService(DaoFactory daoFactory) {
        this.userDao = daoFactory.createUserDao(); // 由工厂创建，不依赖具体实现
    }

    public User getUser(Long id) { return userDao.findById(id); }
}
```
