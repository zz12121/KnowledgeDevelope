# 工厂方法模式（Factory Method）

> 定义一个创建对象的接口，但让子类决定实例化哪个类。工厂方法把对象的实例化推迟到子类。核心思想：把"**创建什么**"延迟到子类，父类只定义"**何时创建**"的流程。

---

## 故事引入

### 《没有中间商赚差价？》

从前有个 Pizza 店，老板娘直接面对顾客。顾客要什么 Pizza 她就做什么，后来订单太多了，老板娘忙不过来，就开始培养徒弟：
- 大徒弟只会做水果Pizza
- 二徒弟只会做肉类Pizza
- 三徒弟只会做蔬菜Pizza

顾客来了，老板娘根据顾客需求让不同徒弟去做——这就是**工厂方法**。

## 一、定义与意图

定义一个创建对象的接口，但让子类决定实例化哪个类。工厂方法把对象的实例化推迟到子类。核心思想：把"**创建什么**"延迟到子类，父类只定义"**何时创建**"的流程。

## 二、类图角色

```
Creator（抽象工厂）
├── + factoryMethod(): Product   // 工厂方法（抽象）
└── + someOperation()            // 使用工厂方法的业务逻辑

ConcreteCreatorA extends Creator
└── + factoryMethod(): ConcreteProductA

ConcreteCreatorB extends Creator
└── + factoryMethod(): ConcreteProductB

Product（产品接口）
ConcreteProductA implements Product
ConcreteProductB implements Product
```

## 三、Demo

```java
// 产品接口
public interface Logger {
    void info(String message);
    void error(String message, Throwable e);
    void warn(String message);
}

// 具体产品：控制台日志
public class ConsoleLogger implements Logger {
    private final String name;
    public ConsoleLogger(String name) { this.name = name; }

    public void info(String message) {
        System.out.printf("[INFO][%s] %s%n", name, message);
    }
    public void error(String message, Throwable e) {
        System.err.printf("[ERROR][%s] %s: %s%n", name, message, e.getMessage());
    }
    public void warn(String message) {
        System.out.printf("[WARN][%s] %s%n", name, message);
    }
}

// 具体产品：文件日志
public class FileLogger implements Logger {
    private final String name;
    private final String filePath;
    public FileLogger(String name, String filePath) { this.name = name; this.filePath = filePath; }

    public void info(String message) { writeToFile("[INFO] " + message); }
    public void error(String message, Throwable e) { writeToFile("[ERROR] " + message); }
    public void warn(String message) { writeToFile("[WARN] " + message); }
    private void writeToFile(String content) { /* 写文件 */ }
}

// 抽象工厂：定义工厂方法
public abstract class LoggerFactory {
    // 工厂方法（子类决定创建哪种 Logger）
    protected abstract Logger createLogger(String name);

    // 模板方法：工厂创建后统一做初始化
    public Logger getLogger(String name) {
        Logger logger = createLogger(name);
        // 可以在这里统一做些初始化（如注册到管理中心）
        return logger;
    }
}

// 具体工厂：控制台日志工厂
public class ConsoleLoggerFactory extends LoggerFactory {
    @Override
    protected Logger createLogger(String name) {
        return new ConsoleLogger(name);
    }
}

// 具体工厂：文件日志工厂
public class FileLoggerFactory extends LoggerFactory {
    private final String logDir;
    public FileLoggerFactory(String logDir) { this.logDir = logDir; }

    @Override
    protected Logger createLogger(String name) {
        return new FileLogger(name, logDir + "/" + name + ".log");
    }
}

// 使用
LoggerFactory factory = new ConsoleLoggerFactory();
// 切换到文件日志只需换工厂，业务代码不变
// LoggerFactory factory = new FileLoggerFactory("/var/logs");

Logger logger = factory.getLogger("OrderService");
logger.info("订单创建成功");
```

## 四、实际业务场景

```java
// 场景：不同渠道的消息发送
public interface MessageSender {
    boolean send(String recipient, String subject, String content);
}

public class EmailSender implements MessageSender {
    private final String smtpHost;
    public EmailSender(String smtpHost) { this.smtpHost = smtpHost; }

    public boolean send(String recipient, String subject, String content) {
        System.out.println("发送邮件到 " + recipient + ": " + subject);
        return true;
    }
}

public class SMSSender implements MessageSender {
    private final String apiKey;
    public SMSSender(String apiKey) { this.apiKey = apiKey; }

    public boolean send(String recipient, String subject, String content) {
        System.out.println("发送短信到 " + recipient + ": " + content);
        return true;
    }
}

public abstract class NotificationFactory {
    protected abstract MessageSender createSender();

    public boolean notify(String recipient, String title, String body) {
        MessageSender sender = createSender();
        return sender.send(recipient, title, body);
    }
}

@Component
public class EmailNotificationFactory extends NotificationFactory {
    @Value("${mail.smtp.host}") private String smtpHost;

    @Override
    protected MessageSender createSender() { return new EmailSender(smtpHost); }
}

@Component
public class SMSNotificationFactory extends NotificationFactory {
    @Value("${sms.api.key}") private String apiKey;

    @Override
    protected MessageSender createSender() { return new SMSSender(apiKey); }
```
