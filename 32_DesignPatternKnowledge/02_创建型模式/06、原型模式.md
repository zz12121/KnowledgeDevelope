# 原型模式（Prototype）

> 通过**克隆已有对象**来创建新对象，而不是通过 new 重新初始化。适合：对象创建成本高（复杂初始化）；需要创建与已有对象差别不大的新对象；避免反复执行相同的初始化逻辑。

---

## 故事引入

### 《小王的学习秘籍》

小王成绩很差学习方法不对，后来他发现学霸的学习笔记非常有效，于是借来复印了一份。他不需要自己重新整理知识点，直接复制学霸的笔记，在此基础上修改成适合自己的版本，就能快速提升成绩。

**原型模式**正是如此——通过克隆已有对象来创建新对象，不需要从零开始初始化。

## 三、Demo

```java
// 游戏角色原型：创建时需要加载大量数据，用克隆避免重复初始化
public class GameCharacter implements Cloneable {
    private String name;
    private String characterClass;    // 职业：warrior/mage/archer
    private int level;
    private int health;
    private int attack;
    private int defense;
    private List<String> skills;      // 技能列表（需要深拷贝）
    private Map<String, Integer> equipment; // 装备（需要深拷贝）
    private List<String> inventory;   // 背包（需要深拷贝）

    public GameCharacter(String characterClass) {
        this.characterClass = characterClass;
        // 模拟复杂的初始化：从数据库/配置加载职业数据
        this.skills = loadSkillsByClass(characterClass);       // 耗时操作
        this.equipment = loadDefaultEquipment(characterClass); // 耗时操作
        this.inventory = new ArrayList<>();
        System.out.println("初始化角色职业: " + characterClass + "（复杂初始化完成）");
    }

    // 深拷贝：复制自己，独立的技能/装备/背包列表
    @Override
    public GameCharacter clone() {
        try {
            GameCharacter copy = (GameCharacter) super.clone();
            copy.skills = new ArrayList<>(this.skills);           // 独立的 List
            copy.equipment = new HashMap<>(this.equipment);       // 独立的 Map
            copy.inventory = new ArrayList<>(this.inventory);     // 独立的背包
            return copy;
        } catch (CloneNotSupportedException e) {
            throw new AssertionError("不应该发生", e);
        }
    }

    // 设置玩家个性化信息（在克隆的副本上修改）
    public GameCharacter withName(String name) {
        this.name = name;
        return this;
    }

    private List<String> loadSkillsByClass(String cls) {
        Map<String, List<String>> skillMap = Map.of(
            "warrior", List.of("重击", "盾墙", "战吼"),
            "mage",    List.of("火球术", "冰封", "魔法护盾"),
            "archer",  List.of("穿心箭", "急速射", "毒箭")
        );
        return new ArrayList<>(skillMap.getOrDefault(cls, List.of()));
    }

    private Map<String, Integer> loadDefaultEquipment(String cls) {
        Map<String, Map<String, Integer>> equipMap = Map.of(
            "warrior", Map.of("剑", 50, "盾", 30),
            "mage",    Map.of("法杖", 80, "法袍", 20),
            "archer",  Map.of("弓", 70, "箭", 200)
        );
        return new HashMap<>(equipMap.getOrDefault(cls, Map.of()));
    }

    @Override
    public String toString() {
        return String.format("角色[%s, 职业=%s, Lv.%d, HP=%d, 技能=%s]",
                name, characterClass, level, health, skills);
    }
}

// 原型注册表：管理各类原型
public class CharacterPrototypeRegistry {
    private static final Map<String, GameCharacter> prototypes = new HashMap<>();

    static {
        // 每种职业只初始化一次（复杂初始化只执行3次）
        prototypes.put("warrior", new GameCharacter("warrior"));
        prototypes.put("mage",    new GameCharacter("mage"));
        prototypes.put("archer",  new GameCharacter("archer"));
    }

    public static GameCharacter createCharacter(String type, String playerName) {
        GameCharacter prototype = prototypes.get(type);
        if (prototype == null) throw new IllegalArgumentException("未知职业: " + type);
        // 克隆原型，设置玩家名称，不需要重新初始化职业数据
        return prototype.clone().withName(playerName);
    }
}

// 使用：创建 1000 个角色，但复杂初始化只执行了 3 次
for (int i = 0; i < 1000; i++) {
    GameCharacter player = CharacterPrototypeRegistry.createCharacter("warrior", "玩家" + i);
    System.out.println(player);
}
```

## 四、实际业务场景

```java
// 场景：报表模板克隆（避免重复构建复杂的报表配置）
public class ReportTemplate implements Cloneable {
    private String templateName;
    private List<ReportColumn> columns;    // 列定义
    private List<ReportFilter> filters;    // 过滤条件
    private ChartConfig chartConfig;       // 图表配置
    private StyleConfig styleConfig;       // 样式配置

    // 深拷贝
    @Override
    public ReportTemplate clone() {
        try {
            ReportTemplate copy = (ReportTemplate) super.clone();
            copy.columns = columns.stream()
                    .map(ReportColumn::clone)
                    .collect(Collectors.toList());
            copy.filters = new ArrayList<>(filters);
            copy.chartConfig = chartConfig.clone();
            copy.styleConfig = styleConfig.clone();
            return copy;
        } catch (CloneNotSupportedException e) {
            throw new AssertionError(e);
        }
    }

    public ReportTemplate withName(String name) {
        this.templateName = name;
        return this;
    }

    public ReportTemplate addFilter(ReportFilter filter) {
        this.filters.add(filter);
        return this;
    }
}

// 使用：基于模板克隆，再做定制
ReportTemplate salesTemplate = templateRegistry.get("sales_monthly");
ReportTemplate customReport = salesTemplate.clone()
        .withName("Q1季度销售报告")
        .addFilter(new DateRangeFilter("2024-01-01", "2024-03-31"));
```
