# 冒泡排序 - 最直观的排序算法

## 🎯 算法思想

### 1.1 直观理解
想象水中的气泡：轻的气泡会慢慢"冒"到水面。在排序中，较大的元素会逐渐"冒"到数组的末尾。

### 1.2 基本思想
通过重复遍历数组，比较相邻元素，如果顺序错误就交换它们，直到整个数组有序。

## 🛠️ 算法步骤

### 2.1 基本冒泡排序
```
原始数组: [5, 2, 8, 1, 9]

第一轮遍历:
比较 5和2 → 交换 → [2, 5, 8, 1, 9]
比较 5和8 → 不交换 → [2, 5, 8, 1, 9]  
比较 8和1 → 交换 → [2, 5, 1, 8, 9]
比较 8和9 → 不交换 → [2, 5, 1, 8, 9]

第二轮遍历:
比较 2和5 → 不交换 → [2, 5, 1, 8, 9]
比较 5和1 → 交换 → [2, 1, 5, 8, 9]
比较 5和8 → 不交换 → [2, 1, 5, 8, 9]

第三轮遍历:
比较 2和1 → 交换 → [1, 2, 5, 8, 9]
比较 2和5 → 不交换 → [1, 2, 5, 8, 9]

第四轮遍历:
比较 1和2 → 不交换 → [1, 2, 5, 8, 9] （已有序）
```

### 2.2 Java实现
```java
public class BubbleSort {
    
    // 基础版本
    public static void bubbleSort(int[] arr) {
        int n = arr.length;
        
        // 外层循环控制遍历轮数
        for (int i = 0; i < n - 1; i++) {
            // 内层循环进行相邻元素比较和交换
            for (int j = 0; j < n - 1 - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    // 交换元素
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }
    
    // 优化版本：提前终止
    public static void bubbleSortOptimized(int[] arr) {
        int n = arr.length;
        boolean swapped;
        
        for (int i = 0; i < n - 1; i++) {
            swapped = false;
            
            for (int j = 0; j < n - 1 - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    // 交换元素
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                    swapped = true;
                }
            }
            
            // 如果这一轮没有发生交换，说明已经有序
            if (!swapped) {
                break;
            }
        }
    }
    
    // 测试代码
    public static void main(String[] args) {
        int[] arr = {64, 34, 25, 12, 22, 11, 90};
        
        System.out.println("原始数组:");
        printArray(arr);
        
        bubbleSortOptimized(arr);
        
        System.out.println("排序后数组:");
        printArray(arr);
    }
    
    public static void printArray(int[] arr) {
        for (int value : arr) {
            System.out.print(value + " ");
        }
        System.out.println();
    }
}
```

## 📊 复杂度分析

### 3.1 时间复杂度
- **最坏情况**：O(n²) - 数组完全逆序
- **最好情况**：O(n) - 数组已经有序（优化版本）
- **平均情况**：O(n²)

### 3.2 空间复杂度
- O(1) - 只需要常数级别的额外空间

### 3.3 稳定性
- **稳定** - 相等元素不会交换位置

## 🔍 算法细节

### 4.1 为什么叫"冒泡"？
因为较大的元素会像气泡一样逐渐"冒"到数组的末尾。

### 4.2 优化策略
1. **提前终止**：如果某一轮没有发生交换，说明已经有序
2. **记录最后交换位置**：下一轮只需遍历到上次最后交换的位置

### 4.3 进一步优化版本
```java
public static void bubbleSortSuperOptimized(int[] arr) {
    int n = arr.length;
    int lastSwapIndex = n - 1;
    
    while (lastSwapIndex > 0) {
        int currentSwapIndex = 0;
        
        for (int j = 0; j < lastSwapIndex; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                currentSwapIndex = j;
            }
        }
        
        lastSwapIndex = currentSwapIndex;
    }
}
```

## 📈 性能对比

### 5.1 与其他排序算法比较
| 算法 | 平均时间复杂度 | 最好情况 | 最坏情况 | 空间复杂度 | 稳定性 |
|------|---------------|----------|----------|------------|--------|
| 冒泡排序 | O(n²) | O(n) | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(n) | O(n²) | O(1) | 稳定 |
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) | 不稳定 |

### 5.2 适用场景
- 小规模数据排序
- 教育目的，理解排序原理
- 几乎已经有序的数据（优化版本）

### 5.3 不适用场景
- 大规模数据排序
- 对性能要求高的场景
