# 二分查找 - O(log n) 的搜索利器

## 🎯 核心思想

在**有序**数组中，每次与中间元素比较，将搜索范围缩小一半，直到找到目标或确认不存在。

```
在 [1, 3, 5, 7, 9, 11, 13] 中查找 7：

初始: left=0, right=6, mid=3, arr[mid]=7 → 找到！

在 [1, 3, 5, 7, 9, 11, 13] 中查找 6：

第1次: left=0, right=6, mid=3, arr[mid]=7 > 6 → right=2
第2次: left=0, right=2, mid=1, arr[mid]=3 < 6 → left=2
第3次: left=2, right=2, mid=2, arr[mid]=5 < 6 → left=3
left > right，返回 -1，未找到。
```

---

## 🛠️ 基础实现（三种模板）

### 模板1：精确查找（最简单）

```java
// 在有序数组中查找 target，返回索引，不存在返回 -1
public int binarySearch(int[] nums, int target) {
    int left = 0, right = nums.length - 1;

    while (left <= right) {  // 注意：<=
        int mid = left + (right - left) / 2;  // 防止溢出！不用 (left+right)/2
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return -1;
}
```

### 模板2：查找左边界（第一个 >= target 的位置）

```java
// 返回第一个 >= target 的索引（即 target 应该插入的位置）
// 如果所有元素都 < target，返回 nums.length
public int lowerBound(int[] nums, int target) {
    int left = 0, right = nums.length;  // 注意：right = length（开区间）

    while (left < right) {  // 注意：< 而非 <=
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid;  // 注意：right = mid（不是 mid-1）
        }
    }
    return left;  // 返回 left
}

// 查找第一个 == target 的索引
public int findFirst(int[] nums, int target) {
    int pos = lowerBound(nums, target);
    if (pos == nums.length || nums[pos] != target) return -1;
    return pos;
}
```

### 模板3：查找右边界（最后一个 <= target 的位置）

```java
// 返回最后一个 <= target 的索引
public int upperBound(int[] nums, int target) {
    int left = 0, right = nums.length;

    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] <= target) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    return left - 1;  // 最后一个 <= target 的位置
}

// 查找最后一个 == target 的索引
public int findLast(int[] nums, int target) {
    int pos = upperBound(nums, target);
    if (pos < 0 || nums[pos] != target) return -1;
    return pos;
}
```

## 🔑 如何写对二分查找？—— 关键细节

| 问题 | 解法 |
|------|------|
| `left <= right` 还是 `left < right`？ | 精确查找用 `<=`；找边界用 `<` |
| `mid = (left+right)/2` 溢出问题 | 始终用 `left + (right-left)/2` |
| `right = mid-1` 还是 `right = mid`？ | 找边界时用 `right = mid`；精确查找用 `right = mid-1` |
| 返回 `left` 还是 `right`？ | 找左边界返回 `left`；找右边界返回 `left-1` |

## 🎯 二分查找的变体

### 1. 搜索插入位置（LeetCode 35）

```java
public int searchInsert(int[] nums, int target) {
    // 即 lowerBound：找第一个 >= target 的位置
    int left = 0, right = nums.length;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) left = mid + 1;
        else right = mid;
    }
    return left;
}
```

### 2. 在旋转数组中查找（LeetCode 33）

```java
public int search(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) return mid;

        if (nums[left] <= nums[mid]) {
            // 左半部分有序
            if (nums[left] <= target && target < nums[mid]) right = mid - 1;
            else left = mid + 1;
        } else {
            // 右半部分有序
            if (nums[mid] < target && target <= nums[right]) left = mid + 1;
            else right = mid - 1;
        }
    }
    return -1;
}
```

### 3. 找峰值元素（LeetCode 162）

```java
public int findPeakElement(int[] nums) {
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] > nums[mid + 1]) right = mid;  // 峰值在左侧（含mid）
        else left = mid + 1;                          // 峰值在右侧
    }
    return left;
}
```

### 4. 二分答案（最重要的进阶用法！）

当问题满足**单调性**（答案越大越满足/不满足某条件），可以对答案二分：

```java
// 分割数组的最大值（LeetCode 410）
// 将数组分成 k 份，使每份最大和尽量小
public int splitArray(int[] nums, int k) {
    int left = Arrays.stream(nums).max().getAsInt();  // 最小可能答案
    int right = Arrays.stream(nums).sum();            // 最大可能答案

    while (left < right) {
        int mid = left + (right - left) / 2;
        if (canSplit(nums, k, mid)) right = mid;  // mid 可行，尝试更小
        else left = mid + 1;
    }
    return left;
}

// 判断每份最大和不超过 maxSum 时能否分成 k 份
private boolean canSplit(int[] nums, int k, int maxSum) {
    int count = 1, currentSum = 0;
    for (int num : nums) {
        if (currentSum + num > maxSum) {
            count++;
            currentSum = num;
        } else {
            currentSum += num;
        }
    }
    return count <= k;
}
```

## 🚀 总结

二分查找的**本质**是利用单调性，每次将搜索空间减半：

1. **有序数组精确查找**：模板1，`left <= right`，`mid ± 1`
2. **查找左/右边界**：模板2/3，`left < right`，更新 `right = mid`
3. **二分答案**：先确定答案范围，再写 `check` 函数判断可行性
4. **记住防溢出**：始终用 `left + (right - left) / 2`

只要看到"有序"或"单调"，首先想到能否用二分查找！
