# 背包问题

## 背包问题概述

背包问题是动态规划中最经典的一类问题，核心是在**容量约束**下选择物品使**价值最大**。

```
有 n 个物品，背包容量为 W
每个物品有重量 weight[i] 和价值 value[i]
问：在不超过背包容量的条件下，如何选择物品使总价值最大？
```

---

## 一、0/1 背包

### 问题描述

每个物品**只能选一次**（0 不选，1 选）。

### 状态定义

`dp[i][j]` = 前 i 个物品，背包容量为 j 时，能装的**最大价值**

### 状态转移

```
对于第 i 个物品（重量 w，价值 v）：
  - 不装：dp[i][j] = dp[i-1][j]
  - 装入：dp[i][j] = dp[i-1][j-w] + v（前提：j >= w）
  
dp[i][j] = max(dp[i-1][j], dp[i-1][j-w] + v)
```

### 二维 DP 实现

```java
public int knapsack01(int W, int[] weights, int[] values) {
    int n = weights.length;
    int[][] dp = new int[n + 1][W + 1];
    
    for (int i = 1; i <= n; i++) {
        int w = weights[i - 1];
        int v = values[i - 1];
        for (int j = 0; j <= W; j++) {
            dp[i][j] = dp[i - 1][j];  // 不装第 i 个物品
            if (j >= w) {
                dp[i][j] = Math.max(dp[i][j], dp[i - 1][j - w] + v);  // 装第 i 个
            }
        }
    }
    return dp[n][W];
}
// 时间复杂度：O(nW)，空间复杂度：O(nW)
```

### 一维 DP 空间优化（滚动数组）

关键：**从右往左**遍历容量，防止同一物品被多次选取

```java
public int knapsack01(int W, int[] weights, int[] values) {
    int n = weights.length;
    int[] dp = new int[W + 1];
    
    for (int i = 0; i < n; i++) {
        int w = weights[i];
        int v = values[i];
        // 必须从右往左，防止重复选取
        for (int j = W; j >= w; j--) {
            dp[j] = Math.max(dp[j], dp[j - w] + v);
        }
    }
    return dp[W];
}
// 时间复杂度：O(nW)，空间复杂度：O(W)
```

**为什么从右往左？**
- `dp[j]` 依赖 `dp[j-w]`（更小的容量）
- 如果从左往右，`dp[j-w]` 已被当前物品更新过，相当于同一物品选了两次
- 从右往左时，`dp[j-w]` 还是上一轮的值，保证每个物品只选一次

### 例题：分割等和子集（LeetCode 416）

**问题**：判断数组是否能分成两个和相等的子集

**转化**：总和为 sum，每个子集和为 sum/2，相当于从数组中选元素，能否凑出 sum/2

```java
public boolean canPartition(int[] nums) {
    int sum = 0;
    for (int num : nums) sum += num;
    if (sum % 2 != 0) return false;
    
    int target = sum / 2;
    boolean[] dp = new boolean[target + 1];
    dp[0] = true;  // 和为 0 一定可以达到（不选任何物品）
    
    for (int num : nums) {
        for (int j = target; j >= num; j--) {
            dp[j] = dp[j] || dp[j - num];
        }
    }
    return dp[target];
}
```

## 二、完全背包

### 问题描述

每个物品**可以选无限次**。

### 与 0/1 背包的区别

只需要将一维 DP 中的遍历顺序改为**从左往右**：

```java
public int knapsackComplete(int W, int[] weights, int[] values) {
    int n = weights.length;
    int[] dp = new int[W + 1];
    
    for (int i = 0; i < n; i++) {
        int w = weights[i];
        int v = values[i];
        // 从左往右，允许重复选取
        for (int j = w; j <= W; j++) {
            dp[j] = Math.max(dp[j], dp[j - w] + v);
        }
    }
    return dp[W];
}
```

**为什么从左往右？**
- 从左往右时，`dp[j-w]` 是当前轮已更新的值，即允许重复选取同一物品
- 这正是完全背包所需要的

### 例题：零钱兑换（LeetCode 322）

**问题**：给定硬币面值数组，凑成 amount，最少需要多少枚硬币？

```java
public int coinChange(int[] coins, int amount) {
    int[] dp = new int[amount + 1];
    Arrays.fill(dp, amount + 1);  // 初始化为不可能的大值
    dp[0] = 0;
    
    for (int coin : coins) {
        for (int j = coin; j <= amount; j++) {
            dp[j] = Math.min(dp[j], dp[j - coin] + 1);
        }
    }
    return dp[amount] > amount ? -1 : dp[amount];
}
```

### 例题：零钱兑换 II（LeetCode 518）

**问题**：凑成 amount，有多少种组合方式？

```java
public int change(int amount, int[] coins) {
    int[] dp = new int[amount + 1];
    dp[0] = 1;  // 金额为 0 有一种方式：什么都不选
    
    for (int coin : coins) {
        for (int j = coin; j <= amount; j++) {
            dp[j] += dp[j - coin];
        }
    }
    return dp[amount];
}
```

## 三、多重背包

### 问题描述

每个物品有**数量限制** count[i]，最多选 count[i] 次。

### 朴素解法（转化为 0/1 背包）

将每个物品展开为 count[i] 个独立物品：

```java
public int multipleKnapsack(int W, int[] weights, int[] values, int[] counts) {
    int n = weights.length;
    int[] dp = new int[W + 1];
    
    for (int i = 0; i < n; i++) {
        int w = weights[i];
        int v = values[i];
        int c = counts[i];
        // 展开为 c 个 0/1 背包
        for (int k = 0; k < c; k++) {
            for (int j = W; j >= w; j--) {
                dp[j] = Math.max(dp[j], dp[j - w] + v);
            }
        }
    }
    return dp[W];
}
```

## 四、背包问题总结

| 类型 | 每个物品选次数 | 遍历容量方向 |
|------|------------|------------|
| 0/1 背包 | 恰好选 0 或 1 次 | **从右到左** |
| 完全背包 | 可选无限次 | **从左到右** |
| 多重背包 | 最多选 k 次 | 展开为 0/1 背包 |

### 求"能否凑成"（boolean）

```java
dp[j] = dp[j] || dp[j - w]
```

### 求"最少数量"（min）

```java
dp[j] = Math.min(dp[j], dp[j - w] + 1)
// 初始化：dp[0]=0，其余初始化为 MAX
```

### 求"最多数量/最大价值"（max）

```java
dp[j] = Math.max(dp[j], dp[j - w] + v)
// 初始化：全部为 0
```

### 求"方案总数"（count）

```java
dp[j] += dp[j - w]
// 初始化：dp[0]=1，其余为 0
```
