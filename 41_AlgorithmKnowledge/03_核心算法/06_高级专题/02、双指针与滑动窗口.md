# 双指针与滑动窗口

## 一、双指针技巧

双指针（Two Pointers）是一种用两个指针遍历数组/链表来解决问题的技巧，通常可以将 O(n²) 的暴力解法优化为 O(n)。

### 双指针类型

| 类型 | 特征 | 适用场景 |
|------|------|----------|
| 对撞指针 | 两端向中间移动 | 有序数组，求和/回文 |
| 快慢指针 | 同向移动，速度不同 | 链表环、找中点 |
| 滑动窗口 | 同向移动，维护区间 | 子数组/子串问题 |

## 二、对撞指针

### 例一：两数之和 II（LeetCode 167）

```java
public int[] twoSum(int[] numbers, int target) {
    int left = 0, right = numbers.length - 1;
    
    while (left < right) {
        int sum = numbers[left] + numbers[right];
        if (sum == target) {
            return new int[]{left + 1, right + 1};
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return new int[]{};
}
```

### 例二：盛最多水的容器（LeetCode 11）

**贪心思路**：面积 = min(height[left], height[right]) × (right - left)，每次移动较矮的那侧

```java
public int maxArea(int[] height) {
    int left = 0, right = height.length - 1;
    int maxWater = 0;
    
    while (left < right) {
        int water = Math.min(height[left], height[right]) * (right - left);
        maxWater = Math.max(maxWater, water);
        
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }
    return maxWater;
}
```

### 例三：三数之和（LeetCode 15）

```java
public List<List<Integer>> threeSum(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    Arrays.sort(nums);
    
    for (int i = 0; i < nums.length - 2; i++) {
        if (i > 0 && nums[i] == nums[i - 1]) continue;  // 去重
        
        int left = i + 1, right = nums.length - 1;
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            if (sum == 0) {
                result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                while (left < right && nums[left] == nums[left + 1]) left++;
                while (left < right && nums[right] == nums[right - 1]) right--;
                left++;
                right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }
    return result;
}
```

## 三、快慢指针

### 例一：链表中环的检测（LeetCode 141）

```java
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;       // 每次走 1 步
        fast = fast.next.next;  // 每次走 2 步
        if (slow == fast) return true;
    }
    return false;
}
```

### 例二：找链表环的入口（LeetCode 142）

```java
public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;
    
    // 第一步：找相遇点
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) break;
    }
    
    if (fast == null || fast.next == null) return null;  // 无环
    
    // 第二步：找环的入口
    // 将 fast 重置到 head，以相同速度走，再次相遇处即为入口
    fast = head;
    while (fast != slow) {
        slow = slow.next;
        fast = fast.next;
    }
    return fast;
}
```

### 例三：链表的中间节点（LeetCode 876）

```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;  // 偶数节点时返回第二个中间节点
}
```

## 四、滑动窗口

滑动窗口适用于**连续子数组/子串**的最值、计数问题。

### 框架

```java
int left = 0, right = 0;
// 窗口状态（哈希表、计数等）

while (right < n) {
    // 1. 扩大窗口：将 nums[right] 加入窗口
    window.add(nums[right]);
    right++;
    
    // 2. 收缩窗口：当窗口不满足条件时
    while (窗口需要收缩) {
        window.remove(nums[left]);
        left++;
    }
    
    // 3. 记录结果
    updateResult();
}
```

### 例一：最小覆盖子串（LeetCode 76）

**问题**：找 s 中包含 t 所有字符的最小子串

```java
public String minWindow(String s, String t) {
    Map<Character, Integer> need = new HashMap<>();
    Map<Character, Integer> window = new HashMap<>();
    
    for (char c : t.toCharArray()) need.merge(c, 1, Integer::sum);
    
    int left = 0, right = 0;
    int valid = 0;  // 满足条件的字符数
    int start = 0, minLen = Integer.MAX_VALUE;
    
    while (right < s.length()) {
        char c = s.charAt(right++);
        if (need.containsKey(c)) {
            window.merge(c, 1, Integer::sum);
            if (window.get(c).equals(need.get(c))) valid++;
        }
        
        while (valid == need.size()) {
            if (right - left < minLen) {
                start = left;
                minLen = right - left;
            }
            char d = s.charAt(left++);
            if (need.containsKey(d)) {
                if (window.get(d).equals(need.get(d))) valid--;
                window.merge(d, -1, Integer::sum);
            }
        }
    }
    return minLen == Integer.MAX_VALUE ? "" : s.substring(start, start + minLen);
}
```

### 例二：无重复字符的最长子串（LeetCode 3）

```java
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> window = new HashMap<>();
    int left = 0, maxLen = 0;
    
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        window.merge(c, 1, Integer::sum);
        
        while (window.get(c) > 1) {
            char d = s.charAt(left++);
            window.merge(d, -1, Integer::sum);
        }
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}
```

### 例三：长度最小的子数组（LeetCode 209）

**问题**：找和 >= target 的最小长度子数组

```java
public int minSubArrayLen(int target, int[] nums) {
    int left = 0, sum = 0, minLen = Integer.MAX_VALUE;
    
    for (int right = 0; right < nums.length; right++) {
        sum += nums[right];
        
        while (sum >= target) {
            minLen = Math.min(minLen, right - left + 1);
            sum -= nums[left++];
        }
    }
    return minLen == Integer.MAX_VALUE ? 0 : minLen;
}
```

## 双指针总结

**何时用对撞指针**：数组有序 + 找满足条件的两个数

**何时用快慢指针**：链表问题 + 找中点/环

**何时用滑动窗口**：连续子数组/子串 + 包含某些元素的最值
