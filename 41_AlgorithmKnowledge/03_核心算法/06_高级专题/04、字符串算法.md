# 字符串算法

## 一、KMP 算法

KMP（Knuth-Morris-Pratt）是字符串匹配的经典算法，解决：在文本串 text 中查找模式串 pattern 的所有出现位置。

### 暴力法的问题

暴力法 O(mn)：每次匹配失败都要从头开始，浪费了已匹配的信息。

### KMP 核心：next 数组（前缀函数）

`next[i]` = pattern[0..i] 中，**最长公共前后缀**的长度

**示例**：pattern = "aabaa"

```
i=0: "a"     → next[0] = 0
i=1: "aa"    → "a"是公共前后缀 → next[1] = 1
i=2: "aab"   → 无公共前后缀 → next[2] = 0
i=3: "aaba"  → "a"是公共前后缀 → next[3] = 1
i=4: "aabaa" → "aa"是公共前后缀 → next[4] = 2
```

### 构建 next 数组

```java
int[] buildNext(String pattern) {
    int m = pattern.length();
    int[] next = new int[m];
    next[0] = 0;
    int k = 0;  // 当前最长前后缀长度
    
    for (int i = 1; i < m; i++) {
        while (k > 0 && pattern.charAt(k) != pattern.charAt(i)) {
            k = next[k - 1];  // 回退
        }
        if (pattern.charAt(k) == pattern.charAt(i)) {
            k++;
        }
        next[i] = k;
    }
    return next;
}
```

### KMP 匹配

```java
public List<Integer> kmpSearch(String text, String pattern) {
    List<Integer> result = new ArrayList<>();
    int n = text.length(), m = pattern.length();
    int[] next = buildNext(pattern);
    int k = 0;  // 已匹配的长度
    
    for (int i = 0; i < n; i++) {
        while (k > 0 && pattern.charAt(k) != text.charAt(i)) {
            k = next[k - 1];
        }
        if (pattern.charAt(k) == text.charAt(i)) {
            k++;
        }
        if (k == m) {
            result.add(i - m + 1);  // 找到一个匹配
            k = next[k - 1];        // 继续寻找下一个
        }
    }
    return result;
}
// 时间复杂度：O(n + m)
```

## 二、回文串算法

### 方法一：中心扩展法（O(n²)）

```java
public String longestPalindrome(String s) {
    int start = 0, maxLen = 1;
    
    for (int i = 0; i < s.length(); i++) {
        // 奇数长度
        int len1 = expandAroundCenter(s, i, i);
        // 偶数长度
        int len2 = expandAroundCenter(s, i, i + 1);
        int len = Math.max(len1, len2);
        
        if (len > maxLen) {
            maxLen = len;
            start = i - (len - 1) / 2;
        }
    }
    return s.substring(start, start + maxLen);
}

private int expandAroundCenter(String s, int left, int right) {
    while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
        left--;
        right++;
    }
    return right - left - 1;
}
```

### 方法二：Manacher 算法（O(n)）

通过插入特殊字符 '#' 统一奇偶回文：

```java
public String longestPalindrome(String s) {
    // 预处理：#a#b#a# 使得所有回文串长度都是奇数
    StringBuilder sb = new StringBuilder("#");
    for (char c : s.toCharArray()) {
        sb.append(c).append('#');
    }
    String t = sb.toString();
    int n = t.length();
    int[] p = new int[n];  // p[i] = 以 i 为中心的回文半径
    int center = 0, right = 0;
    
    for (int i = 0; i < n; i++) {
        if (i < right) {
            p[i] = Math.min(right - i, p[2 * center - i]);
        }
        while (i - p[i] - 1 >= 0 && i + p[i] + 1 < n 
               && t.charAt(i - p[i] - 1) == t.charAt(i + p[i] + 1)) {
            p[i]++;
        }
        if (i + p[i] > right) {
            center = i;
            right = i + p[i];
        }
    }
    
    int maxLen = 0, maxCenter = 0;
    for (int i = 0; i < n; i++) {
        if (p[i] > maxLen) {
            maxLen = p[i];
            maxCenter = i;
        }
    }
    int start = (maxCenter - maxLen) / 2;
    return s.substring(start, start + maxLen);
}
```

## 三、字符串哈希

将字符串映射为数字，快速判断两个子串是否相同（O(1)）。

### Rabin-Karp 算法（滚动哈希）

```java
public int strStr(String haystack, String needle) {
    int n = haystack.length(), m = needle.length();
    if (m == 0) return 0;
    if (n < m) return -1;
    
    long BASE = 31;
    long MOD = 1_000_000_007L;
    
    // 计算 BASE^(m-1) % MOD
    long power = 1;
    for (int i = 0; i < m - 1; i++) power = power * BASE % MOD;
    
    // 计算 needle 的哈希值
    long needleHash = 0;
    for (char c : needle.toCharArray()) {
        needleHash = (needleHash * BASE + (c - 'a' + 1)) % MOD;
    }
    
    // 滑动窗口计算 haystack 的哈希值
    long windowHash = 0;
    for (int i = 0; i < n; i++) {
        windowHash = (windowHash * BASE + (haystack.charAt(i) - 'a' + 1)) % MOD;
        
        if (i >= m) {
            windowHash = (windowHash - (haystack.charAt(i - m) - 'a' + 1) * power % MOD + MOD) % MOD;
        }
        if (i >= m - 1 && windowHash == needleHash) {
            // 哈希相同时验证（避免哈希冲突）
            if (haystack.substring(i - m + 1, i + 1).equals(needle)) {
                return i - m + 1;
            }
        }
    }
    return -1;
}
```

## 四、常见字符串问题

### 同构字符串（LeetCode 205）

```java
public boolean isIsomorphic(String s, String t) {
    Map<Character, Character> st = new HashMap<>();
    Map<Character, Character> ts = new HashMap<>();
    
    for (int i = 0; i < s.length(); i++) {
        char a = s.charAt(i), b = t.charAt(i);
        if (st.containsKey(a) && st.get(a) != b) return false;
        if (ts.containsKey(b) && ts.get(b) != a) return false;
        st.put(a, b);
        ts.put(b, a);
    }
    return true;
}
```

### 字符串轮转（LeetCode 796）

```java
// s 是否是 goal 轮转后的结果
public boolean rotateString(String s, String goal) {
    if (s.length() != goal.length()) return false;
    return (s + s).contains(goal);
}
```

### 字母异位词分组（LeetCode 49）

```java
public List<List<String>> groupAnagrams(String[] strs) {
    Map<String, List<String>> map = new HashMap<>();
    
    for (String str : strs) {
        char[] chars = str.toCharArray();
        Arrays.sort(chars);
        String key = new String(chars);
        map.computeIfAbsent(key, k -> new ArrayList<>()).add(str);
    }
    return new ArrayList<>(map.values());
}
```
