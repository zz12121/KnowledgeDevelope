# 146. LRU 缓存

> 难度：🟡 Medium | 标签：`哈希表` `链表` `设计`

## 题目描述

请你设计并实现一个满足 **LRU（最近最少使用）缓存**约束的数据结构。

实现 `LRUCache` 类：
- `LRUCache(int capacity)` 以**正整数**作为容量 `capacity` 初始化
- `int get(int key)` 如果关键字存在，返回其值，否则返回 `-1`
- `void put(int key, int value)` 插入或更新值。当缓存达到容量时，在插入新项之前，使最长时间未使用的关键字**无效**

`get` 和 `put` 必须以 **O(1)** 时间复杂度运行。

## 示例

```
LRUCache lRUCache = new LRUCache(2);
lRUCache.put(1, 1); // 缓存 {1=1}
lRUCache.put(2, 2); // 缓存 {1=1, 2=2}
lRUCache.get(1);    // 返回 1
lRUCache.put(3, 3); // 缓存满，淘汰 key=2，缓存 {1=1, 3=3}
lRUCache.get(2);    // 返回 -1（已淘汰）
lRUCache.put(4, 4); // 淘汰 key=1，缓存 {4=4, 3=3}
lRUCache.get(1);    // 返回 -1
lRUCache.get(3);    // 返回 3
lRUCache.get(4);    // 返回 4
```

## 解题思路

### 哈希表 + 双向链表 ⭐

- **哈希表**：O(1) 查找 key 对应的节点
- **双向链表**：维护访问顺序，头部最新，尾部最旧；O(1) 删除和插入节点

操作：
- `get(key)`：在哈希表中找到节点，将其移到链表头部，返回值
- `put(key, value)`：
  - 存在：更新值，移到头部
  - 不存在：创建新节点插入头部；若超容量，删除尾部节点并从哈希表移除

## 代码

```java
class LRUCache {
    // 双向链表节点
    private static class Node {
        int key, val;
        Node prev, next;
        Node(int key, int val) {
            this.key = key;
            this.val = val;
        }
    }
    
    private final int capacity;
    private final Map<Integer, Node> map; // key -> 节点
    // 虚拟头尾节点（简化边界处理）
    private final Node head, tail;
    
    public LRUCache(int capacity) {
        this.capacity = capacity;
        map = new HashMap<>();
        head = new Node(0, 0); // 虚拟头（最新）
        tail = new Node(0, 0); // 虚拟尾（最旧）
        head.next = tail;
        tail.prev = head;
    }
    
    public int get(int key) {
        if (!map.containsKey(key)) return -1;
        Node node = map.get(key);
        moveToHead(node); // 访问后移到最新
        return node.val;
    }
    
    public void put(int key, int value) {
        if (map.containsKey(key)) {
            Node node = map.get(key);
            node.val = value;
            moveToHead(node);
        } else {
            Node node = new Node(key, value);
            map.put(key, node);
            addToHead(node);
            if (map.size() > capacity) {
                Node removed = removeTail(); // 淘汰最旧的
                map.remove(removed.key);
            }
        }
    }
    
    // 将节点移到链表头部（最新位置）
    private void moveToHead(Node node) {
        removeNode(node);
        addToHead(node);
    }
    
    private void addToHead(Node node) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }
    
    private void removeNode(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }
    
    private Node removeTail() {
        Node node = tail.prev; // 真正的最后一个节点
        removeNode(node);
        return node;
    }
}
```

## 复杂度分析

| 操作 | 时间 | 空间 |
|------|------|------|
| get / put | O(1) | O(capacity) |
