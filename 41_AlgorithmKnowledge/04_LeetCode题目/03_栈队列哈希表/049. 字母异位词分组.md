# 049. 字母异位词分组

> 难度：🟡 Medium | 标签：`哈希表` `字符串` `排序`

## 题目描述

给你一个字符串数组，请你将**字母异位词**组合在一起。可以按任意顺序返回结果列表。

**字母异位词**：字母相同但排列不同的字符串。

## 示例

```
输入：strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出：[["bat"], ["nat", "tan"], ["ate", "eat", "tea"]]

输入：strs = [""]
输出：[[""]]

输入：strs = ["a"]
输出：[["a"]]
```

## 解题思路

### 方法一：排序作为 Key ⭐

互为字母异位词的字符串，**排序后的结果相同**。

以排序后的字符串作为哈希表的 key，将相同 key 的字符串分组。

### 方法二：字符计数作为 Key

统计每个字符串中各字母出现次数，以计数数组（转为字符串）作为 key。

## 代码

```java
// 方法一：排序 O(n * k log k)
public List<List<String>> groupAnagrams(String[] strs) {
    Map<String, List<String>> map = new HashMap<>();
    
    for (String str : strs) {
        char[] chars = str.toCharArray();
        Arrays.sort(chars); // "eat" -> "aet"，"tea" -> "aet"
        String key = new String(chars);
        
        // 将 str 加入对应分组
        map.computeIfAbsent(key, k -> new ArrayList<>()).add(str);
    }
    return new ArrayList<>(map.values());
}

// 方法二：字符计数 O(n * k)
public List<List<String>> groupAnagrams2(String[] strs) {
    Map<String, List<String>> map = new HashMap<>();
    
    for (String str : strs) {
        int[] count = new int[26];
        for (char c : str.toCharArray()) count[c - 'a']++;
        // 将计数数组转为字符串作为 key，如 "1#0#0#...1#..."
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < 26; i++) {
            sb.append('#').append(count[i]);
        }
        String key = sb.toString();
        map.computeIfAbsent(key, k -> new ArrayList<>()).add(str);
    }
    return new ArrayList<>(map.values());
}
```

## 复杂度分析

| 方法 | 时间 | 空间 |
|------|------|------|
| 排序 | O(n·k log k)，k 为最长字符串长度 | O(n·k) |
| 字符计数 | O(n·k) | O(n·k) |
