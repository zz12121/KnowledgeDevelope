# 053. 最大子数组和

> 难度：🟡 Medium | 标签：`数组` `动态规划` `分治`

## 题目描述

给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

## 示例

```
输入：nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
输出：6
解释：连续子数组 [4, -1, 2, 1] 的和最大，为 6

输入：nums = [1]
输出：1

输入：nums = [5, 4, -1, 7, 8]
输出：23
```

## 解题思路

### 动态规划 O(n) ⭐

**状态**：`dp[i]` = 以 `nums[i]` 结尾的连续子数组的最大和

**转移**：
- 若 `dp[i-1] > 0`：接上前面的子数组有收益 → `dp[i] = dp[i-1] + nums[i]`
- 若 `dp[i-1] <= 0`：前面的子数组是负担，重新开始 → `dp[i] = nums[i]`

合并：`dp[i] = max(dp[i-1] + nums[i], nums[i])`

**最终答案**：`max(dp[0], dp[1], ..., dp[n-1])`

## 代码

```java
// 方法一：动态规划（完整版）
public int maxSubArray(int[] nums) {
    int n = nums.length;
    int[] dp = new int[n];
    dp[0] = nums[0];
    int maxSum = dp[0];
    
    for (int i = 1; i < n; i++) {
        dp[i] = Math.max(dp[i - 1] + nums[i], nums[i]);
        maxSum = Math.max(maxSum, dp[i]);
    }
    return maxSum;
}

// 方法二：空间优化版（推荐）
public int maxSubArray(int[] nums) {
    int maxSum = nums[0];
    int currSum = nums[0];
    
    for (int i = 1; i < nums.length; i++) {
        // 如果前面的累计和是负数，不如抛弃，重新从 nums[i] 开始
        currSum = Math.max(currSum + nums[i], nums[i]);
        maxSum = Math.max(maxSum, currSum);
    }
    return maxSum;
}
```
