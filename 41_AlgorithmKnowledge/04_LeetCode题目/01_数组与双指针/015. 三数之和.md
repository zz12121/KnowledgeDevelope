# 015. 三数之和

> 难度：🟡 Medium | 标签：`数组` `双指针` `排序`

## 题目描述

给你一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k`，同时还满足 `nums[i] + nums[j] + nums[k] == 0`。

请你返回所有和为 `0` 且不重复的三元组。

## 示例

```
输入：nums = [-1, 0, 1, 2, -1, -4]
输出：[[-1, -1, 2], [-1, 0, 1]]

输入：nums = [0, 0, 0]
输出：[[0, 0, 0]]
```

## 解题思路

### 排序 + 双指针 O(n²) ⭐

1. **先排序**：使相同元素相邻，便于去重
2. **固定第一个数** `nums[i]`，问题转化为在剩余数组中找两数之和为 `-nums[i]`
3. 用对撞双指针在 `[i+1, n-1]` 区间内找答案
4. **去重**：跳过相同的 `nums[i]`、`nums[left]`、`nums[right]`

**去重逻辑说明**：
- 外层：`if (i > 0 && nums[i] == nums[i-1]) continue`
- 找到一组后：跳过 left 右边相同的，跳过 right 左边相同的

## 代码

```java
public List<List<Integer>> threeSum(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    Arrays.sort(nums); // 先排序
    
    for (int i = 0; i < nums.length - 2; i++) {
        // 最小值都大于0，不可能有三数之和为0
        if (nums[i] > 0) break;
        // 去重：跳过重复的第一个数
        if (i > 0 && nums[i] == nums[i - 1]) continue;
        
        int left = i + 1, right = nums.length - 1;
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            if (sum == 0) {
                result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                // 去重：跳过重复的 left 和 right
                while (left < right && nums[left] == nums[left + 1]) left++;
                while (left < right && nums[right] == nums[right - 1]) right--;
                left++;
                right--;
            } else if (sum < 0) {
                left++;  // 和太小，left 右移
            } else {
                right--; // 和太大，right 左移
            }
        }
    }
    return result;
}
```

## 复杂度分析

| | 复杂度 |
|---|---|
| 时间 | O(n²)，排序 O(n log n) + 双重循环 O(n²) |
| 空间 | O(log n)，排序递归栈空间 |
