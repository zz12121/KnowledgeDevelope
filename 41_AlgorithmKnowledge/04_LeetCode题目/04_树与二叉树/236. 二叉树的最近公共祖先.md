# 236. 二叉树的最近公共祖先

> 难度：🟡 Medium | 标签：`树` `DFS` `递归`

## 题目描述

给定一个二叉树，找到该树中两个指定节点的最近公共祖先（LCA）。

**最近公共祖先**：对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。

## 示例

```
输入：root = [3, 5, 1, 6, 2, 0, 8, null, null, 7, 4], p = 5, q = 1
        3
       / \
      5   1
     / \ / \
    6  2 0  8
      / \
     7   4

输出：3（节点5和节点1的最近公共祖先是节点3）

输入：同树，p = 5, q = 4
输出：5（节点5是节点4的祖先）
```

## 解题思路

### 递归 DFS ⭐

对于每个节点，递归在左右子树中寻找 p 和 q：

- 如果当前节点是 `null`、`p` 或 `q`：直接返回当前节点
- 递归在左子树找，结果为 `left`
- 递归在右子树找，结果为 `right`
- 若 `left` 和 `right` 都非空：说明 p 和 q 分别在左右子树，当前节点就是 LCA
- 若只有一侧非空：返回非空的那侧

## 代码

```java
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    // 基础情况：空节点或找到 p/q
    if (root == null || root == p || root == q) return root;
    
    // 在左子树中寻找
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    // 在右子树中寻找
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    
    // p 和 q 分别在左右子树 → 当前节点是 LCA
    if (left != null && right != null) return root;
    
    // 只有一侧找到，返回那侧的结果
    return left != null ? left : right;
}
```

## 代码逻辑说明

```
情况1：p 和 q 在不同子树
  left 返回 p（或包含p的LCA）
  right 返回 q（或包含q的LCA）
  → 两侧都不为空，当前节点就是答案

情况2：p 是 q 的祖先（或反之）
  当递归到 p 时，直接返回 p（不再往下找）
  另一侧最终找到 q 后，结果往上传递
  → 最终 p 作为答案返回

情况3：当前子树不包含 p 或 q
  两侧都返回 null
  → 返回 null，告知上层"这里没有"
```
