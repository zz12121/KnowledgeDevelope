# 112. 路径总和

> 难度：🟢 Easy | 标签：`树` `DFS` `BFS` `递归`

## 题目描述

给你二叉树的根节点 `root` 和一个表示目标和的整数 `targetSum`。判断该树中是否存在**根节点到叶子节点**的路径，这条路径上所有节点值相加等于目标和 `targetSum`。

**叶子节点**：没有子节点的节点。

## 示例

```
输入：root = [5, 4, 8, 11, null, 13, 4, 7, 2, null, null, null, 1], targetSum = 22
        5
       / \
      4   8
     /   / \
    11  13   4
   /  \       \
  7    2       1

输出：true
解释：路径 5 → 4 → 11 → 2，和为 22
```

## 解题思路

### DFS 递归 ⭐

每向下走一步，目标和减去当前节点值。到达叶子节点时，判断剩余目标是否为 0。

递归定义：`hasPathSum(node, remain)` = 从 `node` 出发，是否存在到叶子节点路径和为 `remain`

## 代码

```java
// 方法一：DFS 递归
public boolean hasPathSum(TreeNode root, int targetSum) {
    if (root == null) return false;
    
    // 到达叶子节点，检查剩余值
    if (root.left == null && root.right == null) {
        return root.val == targetSum;
    }
    
    // 递归检查左右子树
    int remain = targetSum - root.val;
    return hasPathSum(root.left, remain) || hasPathSum(root.right, remain);
}

// 方法二：BFS 迭代（队列存[节点, 当前路径和]）
public boolean hasPathSum_bfs(TreeNode root, int targetSum) {
    if (root == null) return false;
    
    Queue<TreeNode> nodeQ = new LinkedList<>();
    Queue<Integer> sumQ = new LinkedList<>();
    nodeQ.offer(root);
    sumQ.offer(root.val);
    
    while (!nodeQ.isEmpty()) {
        TreeNode node = nodeQ.poll();
        int currSum = sumQ.poll();
        
        if (node.left == null && node.right == null && currSum == targetSum) {
            return true;
        }
        if (node.left != null) {
            nodeQ.offer(node.left);
            sumQ.offer(currSum + node.left.val);
        }
        if (node.right != null) {
            nodeQ.offer(node.right);
            sumQ.offer(currSum + node.right.val);
        }
    }
    return false;
}
```

| | 复杂度 |
|---|---|
| 时间 | O(n)，最坏遍历所有节点 |
| 空间 | O(h)，h 为树高 |
