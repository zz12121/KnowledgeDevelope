# 876. 链表的中间节点

> 难度：🟢 Easy | 标签：`链表` `双指针`

## 题目描述

给你单链表的头结点 `head`，请你找出并返回链表的**中间结点**。

如果有两个中间结点，则返回**第二个**中间结点。

## 示例

```
输入：head = [1, 2, 3, 4, 5]
输出：[3, 4, 5]（节点值为 3 的节点）

输入：head = [1, 2, 3, 4, 5, 6]
输出：[4, 5, 6]（节点值为 4 的节点，偶数个节点返回第二个中间节点）
```

## 解题思路

### 快慢指针 O(n) ⭐

- `slow` 每次走 1 步，`fast` 每次走 2 步
- 当 `fast` 走到末尾时，`slow` 刚好在中间
- 奇数长度：`fast.next == null` 时停止 → `slow` 在正中间
- 偶数长度：`fast == null` 时停止 → `slow` 在第二个中间节点

## 代码

```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    
    // fast 走两步，slow 走一步
    // fast == null（偶数长度）或 fast.next == null（奇数长度）时停止
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}
```

## 图解

```
奇数 [1, 2, 3, 4, 5]：
初始：slow=1, fast=1
step1：slow=2, fast=3
step2：slow=3, fast=5
step3：fast.next==null，停止，返回 slow=3 ✅

偶数 [1, 2, 3, 4, 5, 6]：
初始：slow=1, fast=1
step1：slow=2, fast=3
step2：slow=3, fast=5
step3：slow=4, fast=null（fast=6.next=null，再走一步就是null）
fast==null，停止，返回 slow=4 ✅
```

| | 复杂度 |
|---|---|
| 时间 | O(n) |
| 空间 | O(1) |
