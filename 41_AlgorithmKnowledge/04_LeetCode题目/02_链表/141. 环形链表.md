# 141. 环形链表

> 难度：🟢 Easy | 标签：`链表` `哈希表` `双指针`

## 题目描述

给你一个链表的头节点 `head`，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。

## 示例

```
输入：head = [3, 2, 0, -4]，pos = 1（尾部连接到下标1的节点）
输出：true

输入：head = [1, 2]，pos = 0
输出：true

输入：head = [1]，pos = -1（无环）
输出：false
```

## 解题思路

### 方法一：哈希表 O(n) 空间 O(n)
将遍历过的节点存入 HashSet，若某节点已在集合中则有环。

### 方法二：快慢指针（Floyd 判圈） O(n) 空间 O(1) ⭐

- `slow` 每次走 1 步，`fast` 每次走 2 步
- 若有环：fast 和 slow 最终会相遇（类比操场跑圈，快的终会追上慢的）
- 若无环：fast 会走到 null

## 代码

```java
// 方法一：哈希表
public boolean hasCycle(ListNode head) {
    Set<ListNode> visited = new HashSet<>();
    ListNode curr = head;
    while (curr != null) {
        if (visited.contains(curr)) return true;
        visited.add(curr);
        curr = curr.next;
    }
    return false;
}

// 方法二：快慢指针（推荐）
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;       // 慢指针：走1步
        fast = fast.next.next;  // 快指针：走2步
        if (slow == fast) return true; // 相遇，有环
    }
    return false; // fast 走到末尾，无环
}
```

## 复杂度分析

| 方法 | 时间 | 空间 |
|------|------|------|
| 哈希表 | O(n) | O(n) |
| 快慢指针 | O(n) | O(1) ✅ |
