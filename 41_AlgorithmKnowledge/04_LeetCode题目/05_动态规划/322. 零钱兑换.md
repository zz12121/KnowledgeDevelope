# 322. 零钱兑换

> 难度：🟡 Medium | 标签：`动态规划` `BFS` `完全背包`

## 题目描述

给你一个整数数组 `coins`，代表不同面额的硬币；以及一个整数 `amount`，代表总金额。

计算凑成总金额所需的**最少的硬币个数**。如果没有任何一种硬币组合能组成总金额，返回 `-1`。

你可以认为每种硬币的数量是**无限的**。

## 示例

```
输入：coins = [1, 5, 10, 25], amount = 41
输出：3（25 + 10 + 5 + 1 = 41，但最优是 25+10+5+1，不对，25+16...）
实际：1 枚25 + 1枚10 + 1枚5 + 1枚1 = 4枚

输入：coins = [1, 2, 5], amount = 11
输出：3（5 + 5 + 1）

输入：coins = [2], amount = 3
输出：-1

输入：coins = [1], amount = 0
输出：0
```

## 解题思路

### 完全背包 DP ⭐

**状态**：`dp[j]` = 凑出金额 `j` 所需的最少硬币数

**转移**：对每种硬币 `coin`，`dp[j] = min(dp[j], dp[j - coin] + 1)`

**初始化**：
- `dp[0] = 0`（凑出 0 不需要任何硬币）
- `dp[1..amount] = amount + 1`（初始化为不可能的大值，相当于∞）

**遍历方向**：完全背包（可重复选取），从左到右

## 代码

```java
// 方法一：动态规划（完全背包）
public int coinChange(int[] coins, int amount) {
    int[] dp = new int[amount + 1];
    Arrays.fill(dp, amount + 1); // 初始化为"无穷大"
    dp[0] = 0;
    
    for (int coin : coins) {
        for (int j = coin; j <= amount; j++) { // 从左往右：完全背包
            dp[j] = Math.min(dp[j], dp[j - coin] + 1);
        }
    }
    return dp[amount] > amount ? -1 : dp[amount];
}

// 方法二：BFS（从 amount=0 开始，每次扩展一枚硬币）
public int coinChange_bfs(int[] coins, int amount) {
    if (amount == 0) return 0;
    boolean[] visited = new boolean[amount + 1];
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(0);
    visited[0] = true;
    int steps = 0;
    
    while (!queue.isEmpty()) {
        steps++;
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int curr = queue.poll();
            for (int coin : coins) {
                int next = curr + coin;
                if (next == amount) return steps;
                if (next < amount && !visited[next]) {
                    visited[next] = true;
                    queue.offer(next);
                }
            }
        }
    }
    return -1;
}
```

## 复杂度分析

| 方法 | 时间 | 空间 |
|------|------|------|
| DP | O(n × amount) | O(amount) |
| BFS | O(n × amount) | O(amount) |
