# 数组详解 - 最基础的数据结构

## 🎯 数组是什么？

### 1.1 直观理解
想象一排连续的房子，每个房子有固定的门牌号（索引），每个房子里可以住一个人（数据）。

### 1.2 正式定义
**数组（Array）** 是一种线性数据结构，用连续的内存空间存储相同类型的元素，通过索引（下标）快速访问。

## 📊 数组的基本特性

### 2.1 内存布局
```
内存地址: 1000  1004  1008  1012  1016
索引:    [0]    [1]    [2]    [3]    [4]
值:      10    20    30    40    50
```

### 2.2 关键特性
- **连续内存**：元素在内存中连续存储
- **固定大小**：创建时确定大小，不能动态改变
- **随机访问**：通过索引直接访问，时间复杂度 O(1)
- **相同类型**：所有元素必须是同一数据类型

## 🛠️ 数组的基本操作

### 3.1 创建数组
**Java中的数组创建：**
```java
// 方式1：声明并初始化
int[] arr1 = new int[5];          // 创建长度为5的int数组，初始值为0

// 方式2：直接初始化
int[] arr2 = {1, 2, 3, 4, 5};    // 创建并初始化

// 方式3：先声明再初始化
int[] arr3;
arr3 = new int[]{6, 7, 8, 9, 10};
```

### 3.2 访问元素
```java
int[] numbers = {10, 20, 30, 40, 50};

// 访问第一个元素
int first = numbers[0];  // 10

// 访问最后一个元素
int last = numbers[numbers.length - 1];  // 50

// 遍历数组
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// 增强for循环（只读遍历）
for (int num : numbers) {
    System.out.println(num);
}
```

### 3.3 修改元素
```java
int[] scores = {85, 90, 78, 92, 88};

// 修改第三个元素
scores[2] = 95;  // 现在数组: [85, 90, 95, 92, 88]

// 批量修改
for (int i = 0; i < scores.length; i++) {
    scores[i] += 5;  // 每个分数加5分
}
```

## ⚡ 操作复杂度分析

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 访问 | O(1) | 通过索引直接访问 |
| 搜索 | O(n) | 需要遍历查找 |
| 插入 | O(n) | 需要移动后续元素 |
| 删除 | O(n) | 需要移动后续元素 |

## 🔄 动态数组（ArrayList）

### 4.1 为什么需要动态数组？
普通数组的大小固定，动态数组可以自动扩容。

### 4.2 ArrayList的实现原理
```java
// 内部使用数组存储
private transient Object[] elementData;

// 扩容机制：当容量不足时，创建新数组并复制数据
private void grow(int minCapacity) {
    int oldCapacity = elementData.length;
    int newCapacity = oldCapacity + (oldCapacity >> 1);  // 1.5倍扩容
    elementData = Arrays.copyOf(elementData, newCapacity);
}
```

### 4.3 ArrayList vs 普通数组
| 特性 | 普通数组 | ArrayList |
|------|----------|-----------|
| 大小 | 固定 | 动态可变 |
| 性能 | 稍快 | 稍慢（有扩容开销） |
| 功能 | 基础 | 丰富的方法 |
| 类型安全 | 需要 | 泛型保证 |

## 🎯 常见算法问题

### 5.1 找出最大值/最小值
```java
public static int findMax(int[] arr) {
    if (arr == null || arr.length == 0) {
        throw new IllegalArgumentException("数组不能为空");
    }
    
    int max = arr[0];
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}
```

### 5.2 数组反转
```java
public static void reverse(int[] arr) {
    int left = 0;
    int right = arr.length - 1;
    
    while (left < right) {
        // 交换元素
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        
        left++;
        right--;
    }
}
```

### 5.3 数组去重
```java
public static int[] removeDuplicates(int[] arr) {
    if (arr == null || arr.length <= 1) {
        return arr;
    }
    
    Arrays.sort(arr);  // 先排序
    
    int uniqueCount = 1;
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] != arr[i - 1]) {
            uniqueCount++;
        }
    }
    
    int[] result = new int[uniqueCount];
    result[0] = arr[0];
    int index = 1;
    
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] != arr[i - 1]) {
            result[index++] = arr[i];
        }
    }
    
    return result;
}
```

## 📈 多维数组

### 6.1 二维数组
```java
// 创建二维数组
int[][] matrix = new int[3][4];  // 3行4列

// 初始化
int[][] grid = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// 访问元素
int value = grid[1][2];  // 第2行第3列: 6

// 遍历二维数组
for (int i = 0; i < grid.length; i++) {
    for (int j = 0; j < grid[i].length; j++) {
        System.out.print(grid[i][j] + " ");
    }
    System.out.println();
}
```

## 🚀 性能优化技巧

### 7.1 缓存友好
由于内存连续性，数组的访问模式对CPU缓存友好。

### 7.2 避免频繁扩容
如果知道大致大小，预先分配足够空间。

### 7.3 使用System.arraycopy
批量复制数据时比循环更快。
