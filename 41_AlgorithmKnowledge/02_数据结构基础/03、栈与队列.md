# 栈与队列 - 受限的线性数据结构

## 🎯 栈（Stack）

### 1.1 直观理解
想象一摞盘子：只能从顶部放入，也只能从顶部取出——这就是栈的核心思想。**后进先出（LIFO：Last In, First Out）**。

### 1.2 栈的核心操作

| 操作 | 含义 | 时间复杂度 |
|------|------|------------|
| push(e) | 入栈（压栈） | O(1) |
| pop() | 出栈（弹栈） | O(1) |
| peek() | 查看栈顶元素 | O(1) |
| isEmpty() | 判断是否为空 | O(1) |

### 1.3 Java 实现

```java
// 方式1：使用 Java 内置 Deque（推荐，官方不推荐 Stack 类）
Deque<Integer> stack = new ArrayDeque<>();
stack.push(1);   // 入栈
stack.push(2);
stack.push(3);
int top = stack.peek();  // 查看栈顶: 3
int val = stack.pop();   // 出栈: 3
System.out.println(stack.isEmpty());  // false

// 方式2：基于数组手动实现
public class ArrayStack {
    private int[] data;
    private int top;
    private int capacity;

    public ArrayStack(int capacity) {
        this.capacity = capacity;
        this.data = new int[capacity];
        this.top = -1;
    }

    public void push(int val) {
        if (top == capacity - 1) throw new RuntimeException("Stack overflow");
        data[++top] = val;
    }

    public int pop() {
        if (isEmpty()) throw new RuntimeException("Stack underflow");
        return data[top--];
    }

    public int peek() {
        if (isEmpty()) throw new RuntimeException("Stack is empty");
        return data[top];
    }

    public boolean isEmpty() {
        return top == -1;
    }

    public int size() {
        return top + 1;
    }
}
```

### 1.4 栈的经典应用

#### 应用1：括号匹配
```java
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') {
            stack.push(c);
        } else {
            if (stack.isEmpty()) return false;
            char top = stack.pop();
            if (c == ')' && top != '(') return false;
            if (c == ']' && top != '[') return false;
            if (c == '}' && top != '{') return false;
        }
    }
    return stack.isEmpty();
}
// LeetCode 20. 有效的括号
```

#### 应用2：单调栈——下一个更大元素
```java
public int[] nextGreaterElement(int[] nums) {
    int n = nums.length;
    int[] result = new int[n];
    Arrays.fill(result, -1);
    Deque<Integer> stack = new ArrayDeque<>();  // 存索引

    for (int i = 0; i < n; i++) {
        // 当前元素比栈顶大，说明找到了栈顶的"下一个更大元素"
        while (!stack.isEmpty() && nums[i] > nums[stack.peek()]) {
            int idx = stack.pop();
            result[idx] = nums[i];
        }
        stack.push(i);
    }
    return result;
}
// LeetCode 496. 下一个更大元素 I
```

#### 应用3：逆波兰表达式（后缀表达式）
```java
public int evalRPN(String[] tokens) {
    Deque<Integer> stack = new ArrayDeque<>();
    for (String token : tokens) {
        if ("+-*/".contains(token)) {
            int b = stack.pop();
            int a = stack.pop();
            switch (token) {
                case "+": stack.push(a + b); break;
                case "-": stack.push(a - b); break;
                case "*": stack.push(a * b); break;
                case "/": stack.push(a / b); break;
            }
        } else {
            stack.push(Integer.parseInt(token));
        }
    }
    return stack.pop();
}
// LeetCode 150. 逆波兰表达式求值
```

#### 应用4：函数调用栈（理解递归）
```
main() 调用 funcA()
  → funcA() 调用 funcB()
    → funcB() 返回（弹出 funcB 栈帧）
  → funcA() 返回（弹出 funcA 栈帧）
main() 继续执行
```
递归的本质就是操作系统帮你维护了一个调用栈。

## 🎯 队列（Queue）

### 2.1 直观理解
想象排队买票：先来的人先买到票——**先进先出（FIFO：First In, First Out）**。

### 2.2 队列的核心操作

| 操作 | 含义 | 时间复杂度 |
|------|------|------------|
| offer(e) | 入队（队尾） | O(1) |
| poll() | 出队（队首） | O(1) |
| peek() | 查看队首元素 | O(1) |
| isEmpty() | 判断是否为空 | O(1) |

### 2.3 Java 实现

```java
// 方式1：使用 LinkedList（最常用）
Queue<Integer> queue = new LinkedList<>();
queue.offer(1);
queue.offer(2);
queue.offer(3);
System.out.println(queue.peek());   // 1（不删除）
System.out.println(queue.poll());   // 1（删除并返回）
System.out.println(queue.size());   // 2

// 方式2：基于循环数组实现（高效）
public class CircularQueue {
    private int[] data;
    private int head, tail, size, capacity;

    public CircularQueue(int capacity) {
        this.capacity = capacity;
        this.data = new int[capacity];
        this.head = 0;
        this.tail = 0;
        this.size = 0;
    }

    public boolean offer(int val) {
        if (size == capacity) return false;
        data[tail] = val;
        tail = (tail + 1) % capacity;  // 循环利用空间
        size++;
        return true;
    }

    public int poll() {
        if (isEmpty()) throw new RuntimeException("Queue is empty");
        int val = data[head];
        head = (head + 1) % capacity;
        size--;
        return val;
    }

    public int peek() {
        if (isEmpty()) throw new RuntimeException("Queue is empty");
        return data[head];
    }

    public boolean isEmpty() { return size == 0; }
    public int size() { return size; }
}
```

### 2.4 双端队列（Deque）

两端都可以入队和出队，是栈和队列的超集。

```java
Deque<Integer> deque = new ArrayDeque<>();
deque.addFirst(1);   // 从头部添加
deque.addLast(2);    // 从尾部添加
deque.peekFirst();   // 查看头部
deque.peekLast();    // 查看尾部
deque.pollFirst();   // 从头部取出
deque.pollLast();    // 从尾部取出
```

### 2.5 优先队列（Priority Queue）

每次出队的是优先级最高的元素（默认最小值）。

```java
// 小顶堆（默认）
PriorityQueue<Integer> minHeap = new PriorityQueue<>();
minHeap.offer(5);
minHeap.offer(1);
minHeap.offer(3);
System.out.println(minHeap.poll());  // 1（最小值）

// 大顶堆（传入比较器）
PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Collections.reverseOrder());
maxHeap.offer(5);
maxHeap.offer(1);
maxHeap.offer(3);
System.out.println(maxHeap.poll());  // 5（最大值）

// 自定义比较器（例：按字符串长度排序）
PriorityQueue<String> pq = new PriorityQueue<>(Comparator.comparingInt(String::length));
```

### 2.6 队列的经典应用

#### 应用1：BFS（广度优先搜索）
```java
// 二叉树按层遍历
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();  // 当前层节点数
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
// LeetCode 102. 二叉树的层序遍历
```

#### 应用2：滑动窗口最大值（单调队列）
```java
public int[] maxSlidingWindow(int[] nums, int k) {
    int n = nums.length;
    int[] result = new int[n - k + 1];
    // 单调递减队列：存索引，队首始终是窗口最大值的索引
    Deque<Integer> deque = new ArrayDeque<>();

    for (int i = 0; i < n; i++) {
        // 移出窗口外的索引
        while (!deque.isEmpty() && deque.peekFirst() < i - k + 1) {
            deque.pollFirst();
        }
        // 维护单调递减（新元素比队尾大，则队尾无用）
        while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i]) {
            deque.pollLast();
        }
        deque.offerLast(i);
        // 窗口形成后记录结果
        if (i >= k - 1) {
            result[i - k + 1] = nums[deque.peekFirst()];
        }
    }
    return result;
}
// LeetCode 239. 滑动窗口最大值
```

## 📊 栈 vs 队列 对比

| 维度 | 栈（Stack） | 队列（Queue） |
|------|------------|--------------|
| 顺序 | 后进先出（LIFO） | 先进先出（FIFO） |
| 操作端 | 一端（栈顶） | 两端（队首/队尾） |
| 典型应用 | 函数调用、括号匹配、DFS | BFS、任务调度、消息队列 |
| Java推荐实现 | `ArrayDeque` | `LinkedList` / `ArrayDeque` |

## 📝 高频面试题

### Q1：用两个栈实现一个队列（LeetCode 232）
```java
class MyQueue {
    Deque<Integer> inStack = new ArrayDeque<>();   // 负责入队
    Deque<Integer> outStack = new ArrayDeque<>();  // 负责出队

    public void push(int x) {
        inStack.push(x);
    }

    public int pop() {
        move();
        return outStack.pop();
    }

    public int peek() {
        move();
        return outStack.peek();
    }

    private void move() {
        // outStack 为空时才将 inStack 的所有元素倒入
        if (outStack.isEmpty()) {
            while (!inStack.isEmpty()) {
                outStack.push(inStack.pop());
            }
        }
    }

    public boolean empty() {
        return inStack.isEmpty() && outStack.isEmpty();
    }
}
```

### Q2：最小栈（LeetCode 155）
```java
class MinStack {
    Deque<Integer> stack = new ArrayDeque<>();
    Deque<Integer> minStack = new ArrayDeque<>();  // 辅助栈，同步存最小值

    public void push(int val) {
        stack.push(val);
        if (minStack.isEmpty() || val <= minStack.peek()) {
            minStack.push(val);
        } else {
            minStack.push(minStack.peek());
        }
    }

    public void pop() {
        stack.pop();
        minStack.pop();
    }

    public int top() { return stack.peek(); }
    public int getMin() { return minStack.peek(); }
}
```

### Q3：用队列实现栈（LeetCode 225）
```java
class MyStack {
    Queue<Integer> queue = new LinkedList<>();

    public void push(int x) {
        queue.offer(x);
        // 把前面所有元素移到刚入队的元素后面，使其成为队首
        for (int i = 0; i < queue.size() - 1; i++) {
            queue.offer(queue.poll());
        }
    }

    public int pop() { return queue.poll(); }
    public int top() { return queue.peek(); }
    public boolean empty() { return queue.isEmpty(); }
}
```

## 🚀 总结

- **栈**：LIFO，适合需要"回溯"、"撤销"、"递归展开"的场景
- **队列**：FIFO，适合需要"按序处理"、"层序遍历"、"广度优先"的场景
- **单调栈/队列**：在 O(n) 时间解决"下一个更大/小元素"、"滑动窗口极值"等问题
- **优先队列**：在 O(log n) 时间获取最值，解决 Top K、任务调度等问题

Java 中推荐始终使用 `ArrayDeque` 替代 `Stack` 类（Stack 有线程同步开销），作为队列时 `ArrayDeque` 也比 `LinkedList` 更高效。
