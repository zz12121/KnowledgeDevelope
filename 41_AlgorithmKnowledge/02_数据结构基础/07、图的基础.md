# 图 - 万物互联的数据结构

## 🎯 图的基本概念

### 1.1 什么是图？
图（Graph）由**顶点（Vertex）** 和**边（Edge）** 组成，用于表示任意对象之间的关系：社交网络（人与人）、地图（城市与道路）、互联网（网页与链接）。

```
    1 --- 2
    |  \  |
    |   \ |
    4 --- 3
```

### 1.2 核心术语

| 术语 | 含义 |
|------|------|
| 顶点（Vertex/Node） | 图中的节点 |
| 边（Edge） | 两个顶点之间的连接 |
| 有向图 | 边有方向（单行道） |
| 无向图 | 边无方向（双行道） |
| 带权图 | 边有权重（距离、费用等） |
| 度（Degree） | 节点的边数；有向图分入度/出度 |
| 路径（Path） | 顶点序列，相邻顶点之间有边 |
| 环（Cycle） | 起点和终点相同的路径 |
| 连通图 | 任意两点间都有路径（无向图） |
| DAG | 有向无环图，拓扑排序的对象 |

## 🏗️ 图的存储方式

### 2.1 邻接矩阵（Adjacency Matrix）

用二维数组 `adj[i][j]` 表示顶点 i 和 j 之间是否有边（或边的权重）。

```java
// 无向图邻接矩阵（n个顶点）
int n = 5;
int[][] adj = new int[n][n];

// 添加边 1-2、1-3、2-4
adj[1][2] = adj[2][1] = 1;
adj[1][3] = adj[3][1] = 1;
adj[2][4] = adj[4][2] = 1;
```

**优缺点：**
- ✅ 查询边是否存在：O(1)
- ❌ 空间：O(V²)，稀疏图浪费严重
- ❌ 遍历所有邻居：O(V)

### 2.2 邻接表（Adjacency List）——最常用

每个顶点维护一个列表，存储与之相邻的顶点。

```java
// 用 List 数组实现
int n = 5;
List<Integer>[] graph = new List[n];
for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();

// 无向图：添加边 1-2
graph[1].add(2);
graph[2].add(1);

// 有向图：添加边 1→2
graph[1].add(2);

// 带权有向图
List<int[]>[] weightedGraph = new List[n];  // [邻居, 权重]
for (int i = 0; i < n; i++) weightedGraph[i] = new ArrayList<>();
weightedGraph[1].add(new int[]{2, 5});  // 1→2，权重5
```

**优缺点：**
- ✅ 空间：O(V + E)，适合稀疏图
- ✅ 遍历邻居：O(度)
- ❌ 查询边是否存在：O(度)

### 2.3 从输入构建图（面试常用模板）

```java
// 典型输入：n 个顶点，edges[][] 表示边
int n = 5;
int[][] edges = {{0,1},{0,2},{1,3},{2,4}};

// 构建邻接表
List<List<Integer>> graph = new ArrayList<>();
for (int i = 0; i < n; i++) graph.add(new ArrayList<>());
for (int[] edge : edges) {
    graph.get(edge[0]).add(edge[1]);
    graph.get(edge[1]).add(edge[0]);  // 无向图加这行
}
```

## 🔍 图的遍历

### 3.1 深度优先搜索（DFS）

沿着一条路走到底，回溯后再走其他路。

```java
// DFS 递归版本
boolean[] visited;

public void dfs(List<List<Integer>> graph, int node) {
    visited[node] = true;
    System.out.println("访问节点: " + node);

    for (int neighbor : graph.get(node)) {
        if (!visited[neighbor]) {
            dfs(graph, neighbor);
        }
    }
}

// DFS 迭代版本（用栈）
public void dfsIterative(List<List<Integer>> graph, int start, int n) {
    boolean[] visited = new boolean[n];
    Deque<Integer> stack = new ArrayDeque<>();
    stack.push(start);

    while (!stack.isEmpty()) {
        int node = stack.pop();
        if (visited[node]) continue;
        visited[node] = true;
        System.out.println("访问节点: " + node);
        for (int neighbor : graph.get(node)) {
            if (!visited[neighbor]) stack.push(neighbor);
        }
    }
}
```

### 3.2 广度优先搜索（BFS）

按层扩展，先访问近邻，再访问远邻。

```java
// BFS（用队列）
public void bfs(List<List<Integer>> graph, int start, int n) {
    boolean[] visited = new boolean[n];
    Queue<Integer> queue = new LinkedList<>();

    queue.offer(start);
    visited[start] = true;

    while (!queue.isEmpty()) {
        int node = queue.poll();
        System.out.println("访问节点: " + node);

        for (int neighbor : graph.get(node)) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.offer(neighbor);
            }
        }
    }
}

// BFS 求最短路径（无权图）
public int shortestPath(List<List<Integer>> graph, int start, int end, int n) {
    boolean[] visited = new boolean[n];
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(start);
    visited[start] = true;
    int distance = 0;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int node = queue.poll();
            if (node == end) return distance;
            for (int neighbor : graph.get(node)) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);
                }
            }
        }
        distance++;
    }
    return -1;  // 不可达
}
```

## 🔗 并查集（Union-Find）

高效处理**连通性**问题：判断两个节点是否在同一连通分量，合并两个分量。

```java
public class UnionFind {
    private int[] parent;
    private int[] rank;  // 按秩合并
    private int count;   // 连通分量数

    public UnionFind(int n) {
        parent = new int[n];
        rank = new int[n];
        count = n;
        for (int i = 0; i < n; i++) parent[i] = i;
    }

    // 查找根节点（路径压缩）
    public int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]);  // 路径压缩：直接指向根节点
        }
        return parent[x];
    }

    // 合并两个分量
    public boolean union(int x, int y) {
        int rootX = find(x), rootY = find(y);
        if (rootX == rootY) return false;  // 已在同一分量

        // 按秩合并：小树挂到大树下
        if (rank[rootX] < rank[rootY]) parent[rootX] = rootY;
        else if (rank[rootX] > rank[rootY]) parent[rootY] = rootX;
        else { parent[rootY] = rootX; rank[rootX]++; }

        count--;
        return true;
    }

    public boolean connected(int x, int y) {
        return find(x) == find(y);
    }

    public int getCount() { return count; }
}

// 使用示例：岛屿数量（LeetCode 200）变体
// 每个格子是一个节点，相邻的陆地格子 union 在一起
```

## 🎯 拓扑排序

### 5.1 什么是拓扑排序？

对有向无环图（DAG）进行排序，使得所有边 u→v 中，u 排在 v 之前。用于：**课程先修关系、任务依赖、编译顺序**。

### 5.2 Kahn 算法（BFS版，推荐）

```java
public int[] topologicalSort(int n, int[][] prerequisites) {
    List<List<Integer>> graph = new ArrayList<>();
    int[] inDegree = new int[n];

    for (int i = 0; i < n; i++) graph.add(new ArrayList<>());
    for (int[] pre : prerequisites) {
        graph.get(pre[1]).add(pre[0]);  // pre[1] → pre[0]
        inDegree[pre[0]]++;
    }

    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < n; i++) {
        if (inDegree[i] == 0) queue.offer(i);  // 入度为0的节点先入队
    }

    int[] order = new int[n];
    int idx = 0;

    while (!queue.isEmpty()) {
        int node = queue.poll();
        order[idx++] = node;
        for (int neighbor : graph.get(node)) {
            inDegree[neighbor]--;
            if (inDegree[neighbor] == 0) queue.offer(neighbor);
        }
    }

    return idx == n ? order : new int[0];  // 有环则无法完成拓扑排序
}
// LeetCode 207. 课程表 / 210. 课程表 II
```

## 🗺️ 最短路径算法

### 6.1 Dijkstra 算法（非负权图的单源最短路径）

```java
public int[] dijkstra(int n, int[][] edges, int start) {
    // 构建邻接表
    List<int[]>[] graph = new List[n];
    for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
    for (int[] e : edges) {
        graph[e[0]].add(new int[]{e[1], e[2]});  // [邻居, 权重]
    }

    int[] dist = new int[n];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[start] = 0;

    // 优先队列：[距离, 节点]
    PriorityQueue<int[]> pq = new PriorityQueue<>(Comparator.comparingInt(a -> a[0]));
    pq.offer(new int[]{0, start});

    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        int d = curr[0], node = curr[1];

        if (d > dist[node]) continue;  // 已经找到更短路径，跳过

        for (int[] neighbor : graph[node]) {
            int nextNode = neighbor[0], weight = neighbor[1];
            int newDist = dist[node] + weight;
            if (newDist < dist[nextNode]) {
                dist[nextNode] = newDist;
                pq.offer(new int[]{newDist, nextNode});
            }
        }
    }
    return dist;
    // 时间复杂度：O((V + E) log V)
}
// LeetCode 743. 网络延迟时间
```

## 📝 高频面试题

### Q1：岛屿数量（LeetCode 200）
```java
public int numIslands(char[][] grid) {
    int count = 0;
    for (int i = 0; i < grid.length; i++) {
        for (int j = 0; j < grid[0].length; j++) {
            if (grid[i][j] == '1') {
                dfs(grid, i, j);
                count++;
            }
        }
    }
    return count;
}

private void dfs(char[][] grid, int i, int j) {
    if (i < 0 || i >= grid.length || j < 0 || j >= grid[0].length
        || grid[i][j] != '1') return;
    grid[i][j] = '0';  // 标记已访问
    dfs(grid, i+1, j); dfs(grid, i-1, j);
    dfs(grid, i, j+1); dfs(grid, i, j-1);
}
```

### Q2：判断图中是否有环（有向图）
```java
public boolean hasCycle(int n, int[][] edges) {
    List<List<Integer>> graph = new ArrayList<>();
    for (int i = 0; i < n; i++) graph.add(new ArrayList<>());
    for (int[] e : edges) graph.get(e[0]).add(e[1]);

    // 0=未访问，1=访问中，2=已完成
    int[] state = new int[n];
    for (int i = 0; i < n; i++) {
        if (state[i] == 0 && dfsCycle(graph, i, state)) return true;
    }
    return false;
}

private boolean dfsCycle(List<List<Integer>> graph, int node, int[] state) {
    state[node] = 1;  // 标记为访问中
    for (int neighbor : graph.get(node)) {
        if (state[neighbor] == 1) return true;  // 遇到访问中的节点 = 有环
        if (state[neighbor] == 0 && dfsCycle(graph, neighbor, state)) return true;
    }
    state[node] = 2;  // 标记为已完成
    return false;
}
```

### Q3：克隆图（LeetCode 133）
```java
public Node cloneGraph(Node node) {
    if (node == null) return null;
    Map<Node, Node> visited = new HashMap<>();
    return dfs(node, visited);
}

private Node dfs(Node node, Map<Node, Node> visited) {
    if (visited.containsKey(node)) return visited.get(node);
    Node clone = new Node(node.val);
    visited.put(node, clone);
    for (Node neighbor : node.neighbors) {
        clone.neighbors.add(dfs(neighbor, visited));
    }
    return clone;
}
```

## 📊 算法选型指南

| 问题类型 | 推荐算法 | 时间复杂度 |
|----------|----------|------------|
| 连通性判断 | 并查集 / BFS / DFS | O(E·α(V)) / O(V+E) |
| 最短路径（无权） | BFS | O(V+E) |
| 最短路径（非负权） | Dijkstra | O((V+E)log V) |

| 最小生成树 | Kruskal / Prim | O(E log E) |

## 🚀 总结

图是最通用的数据结构，几乎所有关系型问题都可以建模为图：

1. **存储**：邻接表（稀疏图首选），邻接矩阵（查边快）
2. **遍历**：DFS（递归，有回溯）用于连通性/环检测/拓扑排序；BFS 用于最短路径
3. **并查集**：判断连通性的利器，时间复杂度近乎 O(1)
4. **拓扑排序**：课程安排、任务依赖的标准解法（Kahn算法）
5. **Dijkstra**：带权图最短路径，配合优先队列

面试中图题 80% 都是 BFS/DFS，先掌握这两个，其余按需学习。
