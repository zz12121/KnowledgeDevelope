# 树与二叉树 - 层次化数据结构

## 🎯 树的基本概念

### 1.1 什么是树？
树是一种**非线性**数据结构，模拟有层次关系的数据（文件系统、组织架构、DOM 结构等）。

```
          1          ← 根节点（Root）
        /   \
       2     3       ← 内部节点
      / \     \
     4   5     6     ← 叶节点（Leaf，没有子节点）
```

### 1.2 核心术语

| 术语 | 含义 |
|------|------|
| 根节点（Root） | 树的第一个节点，没有父节点 |
| 父节点（Parent） | 有子节点的节点 |
| 子节点（Child） | 有父节点的节点 |
| 叶节点（Leaf） | 没有子节点的节点 |
| 兄弟节点（Sibling） | 拥有同一父节点的节点 |
| 深度（Depth） | 从根节点到该节点的边数 |
| 高度（Height） | 从该节点到最远叶节点的边数 |
| 层（Level） | 深度 + 1 |
| 度（Degree） | 节点的子节点数量 |

### 1.3 树的重要性质
- **N 个节点的树有 N-1 条边**
- 任意两个节点间有唯一路径

## 🌲 二叉树（Binary Tree）

每个节点**最多有 2 个子节点**（左子节点、右子节点）。

### 2.1 节点定义

```java
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}
```

### 2.2 二叉树的种类

```
满二叉树（Full）：每个节点有 0 或 2 个子节点
       1
      / \
     2   3
    / \
   4   5

完全二叉树（Complete）：除最后一层外每层全满，最后一层从左填满
       1
      / \
     2   3
    / \  /
   4   5 6

完美二叉树（Perfect）：所有内部节点都有两个子节点，所有叶节点在同一层
       1
      / \
     2   3
    / \ / \
   4  5 6  7

平衡二叉树（Balanced）：任意节点的左右子树高度差 ≤ 1（AVL 树是典型）
```

## 🔄 二叉树的遍历（最核心！）

### 3.1 深度优先遍历（DFS）

记忆口诀：**根的位置**决定遍历顺序名称
- **前序（Pre-order）**：根 → 左 → 右（根在前）
- **中序（In-order）**：左 → 根 → 右（根在中）
- **后序（Post-order）**：左 → 右 → 根（根在后）

```java
// 前序遍历 - 递归（最直观）
public void preOrder(TreeNode root, List<Integer> result) {
    if (root == null) return;
    result.add(root.val);      // 访问根
    preOrder(root.left, result);   // 遍历左子树
    preOrder(root.right, result);  // 遍历右子树
}

// 中序遍历 - 递归
public void inOrder(TreeNode root, List<Integer> result) {
    if (root == null) return;
    inOrder(root.left, result);    // 遍历左子树
    result.add(root.val);      // 访问根
    inOrder(root.right, result);   // 遍历右子树
}

// 后序遍历 - 递归
public void postOrder(TreeNode root, List<Integer> result) {
    if (root == null) return;
    postOrder(root.left, result);  // 遍历左子树
    postOrder(root.right, result); // 遍历右子树
    result.add(root.val);      // 访问根
}
```

**迭代版本（面试常考）：**
```java
// 前序遍历 - 迭代（用栈模拟递归）
public List<Integer> preOrderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;

    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(node.val);
        // 先压右再压左，保证左先出
        if (node.right != null) stack.push(node.right);
        if (node.left != null) stack.push(node.left);
    }
    return result;
}

// 中序遍历 - 迭代（最经典写法）
public List<Integer> inOrderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode curr = root;

    while (curr != null || !stack.isEmpty()) {
        // 一路向左，全部压栈
        while (curr != null) {
            stack.push(curr);
            curr = curr.left;
        }
        // 弹出，访问，转向右子树
        curr = stack.pop();
        result.add(curr.val);
        curr = curr.right;
    }
    return result;
}

// 后序遍历 - 迭代（前序变形：根右左 → 逆序 = 左右根）
public List<Integer> postOrderIterative(TreeNode root) {
    LinkedList<Integer> result = new LinkedList<>();
    if (root == null) return result;

    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.addFirst(node.val);  // 头插
        if (node.left != null) stack.push(node.left);
        if (node.right != null) stack.push(node.right);
    }
    return result;
}
```

### 3.2 广度优先遍历（BFS）—— 层序遍历

```java
// 层序遍历（LeetCode 102）
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size();  // 当前层的节点数
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

### 3.3 通用递归解题框架

```java
// 二叉树问题万能框架：分解问题 or 遍历
// 框架1：遍历框架（回溯思想）
void traverse(TreeNode root) {
    // 前序位置（进入节点时）
    traverse(root.left);
    // 中序位置（左子树遍历完后）
    traverse(root.right);
    // 后序位置（离开节点时，可以拿到子树的返回值）
}

// 框架2：分解问题（动态规划思想）
// 以"二叉树的最大深度"为例
public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    int leftDepth = maxDepth(root.left);    // 左子树最大深度
    int rightDepth = maxDepth(root.right);  // 右子树最大深度
    return Math.max(leftDepth, rightDepth) + 1;  // 当前节点深度
}
```

## 🔍 二叉搜索树（BST）

### 4.1 定义
- 左子树所有节点 < 根节点
- 右子树所有节点 > 根节点
- 左右子树也分别是 BST

**关键性质**：BST 的**中序遍历结果是有序递增序列**！

### 4.2 BST 基本操作

```java
// 搜索
public TreeNode searchBST(TreeNode root, int val) {
    if (root == null || root.val == val) return root;
    return val < root.val ? searchBST(root.left, val)
                          : searchBST(root.right, val);
    // 时间复杂度：O(h)，h 为树高。平衡时 O(log n)，退化时 O(n)
}

// 插入
public TreeNode insertIntoBST(TreeNode root, int val) {
    if (root == null) return new TreeNode(val);
    if (val < root.val) root.left = insertIntoBST(root.left, val);
    else root.right = insertIntoBST(root.right, val);
    return root;
}

// 删除（三种情况）
public TreeNode deleteNode(TreeNode root, int key) {
    if (root == null) return null;
    if (key < root.val) {
        root.left = deleteNode(root.left, key);
    } else if (key > root.val) {
        root.right = deleteNode(root.right, key);
    } else {
        // 找到了要删除的节点
        if (root.left == null) return root.right;   // 情况1：无左子树
        if (root.right == null) return root.left;   // 情况2：无右子树
        // 情况3：有两个子节点，用右子树最小值（后继节点）替换
        TreeNode minNode = getMin(root.right);
        root.val = minNode.val;
        root.right = deleteNode(root.right, minNode.val);
    }
    return root;
}

private TreeNode getMin(TreeNode node) {
    while (node.left != null) node = node.left;
    return node;
}
```

## 📚 高频面试题

### Q1：对称二叉树（LeetCode 101）
```java
public boolean isSymmetric(TreeNode root) {
    return check(root.left, root.right);
}

private boolean check(TreeNode left, TreeNode right) {
    if (left == null && right == null) return true;
    if (left == null || right == null) return false;
    return left.val == right.val
        && check(left.left, right.right)
        && check(left.right, right.left);
}
```

### Q2：路径总和（LeetCode 112）
```java
public boolean hasPathSum(TreeNode root, int targetSum) {
    if (root == null) return false;
    if (root.left == null && root.right == null) return root.val == targetSum;
    return hasPathSum(root.left, targetSum - root.val)
        || hasPathSum(root.right, targetSum - root.val);
}
```

### Q3：二叉树的最近公共祖先（LeetCode 236）
```java
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    if (left != null && right != null) return root;  // p 和 q 分别在两侧
    return left != null ? left : right;  // 都在同侧
}
```

### Q4：从前序与中序遍历序列构造二叉树（LeetCode 105）
```java
public TreeNode buildTree(int[] preorder, int[] inorder) {
    Map<Integer, Integer> inMap = new HashMap<>();
    for (int i = 0; i < inorder.length; i++) inMap.put(inorder[i], i);
    return build(preorder, 0, preorder.length - 1,
                 inorder, 0, inorder.length - 1, inMap);
}

private TreeNode build(int[] pre, int preL, int preR,
                       int[] in, int inL, int inR,
                       Map<Integer, Integer> inMap) {
    if (preL > preR) return null;
    int rootVal = pre[preL];
    int inRootIdx = inMap.get(rootVal);
    int leftSize = inRootIdx - inL;

    TreeNode root = new TreeNode(rootVal);
    root.left = build(pre, preL + 1, preL + leftSize,
                      in, inL, inRootIdx - 1, inMap);
    root.right = build(pre, preL + leftSize + 1, preR,
                       in, inRootIdx + 1, inR, inMap);
    return root;
}
```

### Q5：验证二叉搜索树（LeetCode 98）
```java
// 利用 BST 中序遍历有序的性质
public boolean isValidBST(TreeNode root) {
    long prev = Long.MIN_VALUE;
    // 中序遍历，检查是否严格递增
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode curr = root;
    while (curr != null || !stack.isEmpty()) {
        while (curr != null) { stack.push(curr); curr = curr.left; }
        curr = stack.pop();
        if (curr.val <= prev) return false;
        prev = curr.val;
        curr = curr.right;
    }
    return true;
}
```

## 📊 常见树的复杂度对比

| 操作 | 普通二叉树 | BST（平衡） | BST（退化） |
|------|-----------|------------|------------|
| 搜索 | O(n) | O(log n) | O(n) |
| 插入 | O(n) | O(log n) | O(n) |
| 删除 | O(n) | O(log n) | O(n) |

> **BST 退化问题**：如果插入有序数据（1,2,3,4...），BST 退化为链表，高度为 n。
> 解决方案：使用**自平衡 BST**，如 AVL 树或红黑树，Java 的 TreeMap/TreeSet 底层就是红黑树。

## 🚀 总结

二叉树是面试中出现频率**极高**的数据结构，核心要掌握：

1. **四种遍历**（前/中/后序 + 层序）的递归和迭代写法
2. **递归框架**：遍历 vs 分解问题，在合适的位置（前/中/后序）写逻辑
3. **BST 特性**：中序有序，搜索/插入/删除均 O(log n)（平衡时）
4. **高频题型**：路径问题、公共祖先、构造树、BST验证/操作
5. 碰到任何二叉树题，先画图，然后思考用**递归**如何分解
